REASSEMBLY SERVICE POINTS


>>A<< OIL SEAL INSTALLATION



1. Apply specified grease to the lip of the oil seal.
Specified lubricant:
Mitsubishi Part No. 0101011 or equivalent
2. Use socket wrench to install the oil seal.

>>B<< OIL SEAL INSTALLATION



1. Apply specified grease to the lip of the oil seal.
Specified lubricant:
Mitsubishi Part No. 0101011 or equivalent
2. Use special tools to install the oil seal.

  • Handle (MB990938)
  • Installer adapter (MB990935)

>>C<< SEAL BOLT INSTALLATION



1. Apply the specified sealant to the threads.
Specified lubricant:
3M STUD Locking No. 4170 or equivalent
noteThe new bolt does not require sealant application as it is precoated with sealant.
