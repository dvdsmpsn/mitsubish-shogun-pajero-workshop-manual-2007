Inspection Procedure 5: Hunting (Repeated Acceleration and Deceleration) Occurs at the Set Vehicle Speed <6G7>.


M11702000043ENG0000010001


COMMENTS ON TROUBLE SYMPTOM


The vehicle speed sensor or the throttle body is suspected.

PROBABLE CAUSES



  • Malfunction of the vehicle speed sensor.
  • Malfunction of the throttle body.
  • Malfunction of the engine-A/T-ECU.

DIAGNOSIS PROCEDURE


STEP 1. Check the MPI system diagnosis code.


Refer to GROUP 13Ab - Diagnosis Function .

Q. Is any diagnosis code set?

Repair the MPI system. (Refer to GROUP 13Ab - Inspection Chart for Diagnosis Code ). Then go to Step 3.
Go to Step 2.

STEP 2. Retest the system


Q. Does a hunting occur?

Replace the engine-A/T-ECU (Refer to GROUP 13Aa - Engine-A/T-ECU ). Then go to Step 3.
It can be assumed that this malfunction is intermittent (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunctions ).

STEP 3. Retest the system


Q. Does a hunting occur?

Return to Step 1.
The procedure is complete.
