STABILIZER LINK BALL JOINT TURNING TORQUE CHECK


M13401000919ENG0000010000



1. After shaking the ball joint stud several times, install the nut to the stud and use special tool preload socket (MB990326) to measure the turning torque of the ball joint.
Standard value: 6.86 - 19.6 N·m
2. If the measured value exceeds the standard value, replace the stabilizer link.
3. If the measured value is lower than the standard value, check that the ball joint turns smoothly without excessive play. If so, it is possible to use that ball joint.
