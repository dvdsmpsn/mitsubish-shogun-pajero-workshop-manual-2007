Inspection Procedure 4: Vehicle speed-dependent unfolding function does not work normally.


M15102200009ENG0000010002


cautionWhenever the ECU is replaced, ensure that the input and output signal circuits are normal.


COMMENTS ON TROUBLE SYMPTOM


The ETACS-ECU operates this function in accordance with the input signals below.

  • Ignition switch (IG1)
  • Vehicle speed signal
If this function does not work normally, these input signal circuit(s) or the ETACS-ECU may be defective. In addition, it is possible that the function except the vehicle speed-dependent unfolding function has been set by using the customized function.

POSSIBLE CAUSES



  • Malfunction of ETACS-ECU
  • Damaged harness wires and connectors

DIAGNOSTIC PROCEDURE


Step 1. Check the electric folding door mirrors operation.


Check that the door mirror is folded and unfolded with the remote controlled mirror switch when the ignition switch is at ACC position.

Q. Is the check result normal?

Go to Step 2.
Refer to inspection procedure 1 "Electric retractable remote controlled door mirrors does not work ."

Step 2. ECU check by using the SWS monitor.


Check that the power supply and earth lines to the ETACS-ECU and the SWS communication lines are normal.

  • Ignition switch: ON
ECU TO BE CHECKED
  • ETACS ECU
OK: "OK" is displayed on the "ETACS ECU" menu.

Q. Is the check result normal?

Go to Step 3.
Refer to GROUP 54Ad, ETACS - Inspection Procedure A-1 "Malfunction of ETACS-ECU power supply circuit ."

Step 3. ECU check by using the SWS monitor.


Check that the vehicle speed signal to the ETACS-ECU is normal.

  • Ignition switch: ON
ECU TO BE CHECKED
  • ETACS ECU
OK: "OK" is displayed on the "ETACS ECU" menu.

Q. Is the check result normal?

Go to Step 4.
Refer to GROUP 54Ad, ETACS - Inspection procedure B-2 "Door ajar warning function does not work normally (combination meter (speedometer) operation check) ."

STEP 4. Retest the system.


Check that the vehicle speed-dependent unfolding function works normally.

Q. Is the check result normal?

The trouble can be an intermittent malfunction (Refer to GROUP 00 - How to Cope with Intermittent Malfunction ).
Replace the ETACS-ECU.
