Code No.U1109 ETACS-ECU Time-out


M15410800386ENG0000010000


caution
  • If diagnosis code U is set in the ETACS-ECU, diagnose the CAN main bus line.
  • Whenever the ECU is replaced, ensure that the communication circuit is normal.

TROUBLE JUDGMENT


The centre display unit or multi display station communicates with the ETACS-ECU via CAN bus lines. If it can not receive all the necessary signals from the ETACS-ECU, diagnosis code U1109 is set.

PROBABLE CAUSES



  • Damaged harness wires and connectors
  • Malfunction of ETACS-ECU
  • Malfunction of centre display unit or multi display station

DIAGNOSIS PROCEDURE


STEP 1. M.U.T.-III CAN bus diagnostics

Use the M.U.T.-III to diagnose the CAN bus lines.

Q. Is the check result satisfactory?

Go to Step 2.
Repair the CAN bus lines. (Refer to GROUP 54D - Troubleshooting ). Repair the CAN bus line and go to Step 4.
STEP 2. M.U.T.-III other system diagnosis code

Check that the ETACS-ECU sets a diagnosis code.

Q. Is the diagnosis code set?

Diagnose the SWS. (Refer to GROUP 54C - Troubleshooting <not using SWS monitor> or <using SWS monitor.)
Go to Step 3.
STEP 3. M.U.T.-III other system diagnosis code

Check whether combination meter-related diagnosis code U1109 is set.

Q. Is the diagnosis code set?

Replace ETACS-ECU and then go to step 4.
Replace centre display unit or multi display station and then go to step 4.
STEP 4. Check whether the diagnosis code is reset.

Recheck if the diagnosis code is set.
(1)Erase the diagnosis code.
(2)Turn the ignition switch to "ON" position.
(3)Check if the diagnosis code is set.

Q. Is the check result satisfactory?

The procedure is complete.
Go to Step 1.
