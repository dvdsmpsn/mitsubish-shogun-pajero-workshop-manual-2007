Code No.12: KEY IDs are not the same or are not registered.


M15410700030ENG0000010000


DIAGNOSIS CODE SET CONDITIONS


This code is set when the ignition key is not registered with the immobilizer-ECU.

PROBABLE CAUSES



  • The ignition key is not registered with the immobilizer-ECU.
  • Malfunction of immobilizer-ECU

DIAGNOSIS PROCEDURE


Resister the key ID.

Q. Was the key ID registered?

The procedure is complete.
Replace the immobilizer-ECU and register the key ID (Refer to ).
