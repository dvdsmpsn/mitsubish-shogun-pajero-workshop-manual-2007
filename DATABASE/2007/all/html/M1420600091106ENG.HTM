INSPECTION PROCEDURE 5: The Sunroof Inhibition Function does not work Normally (The Sunroof Operation Lock Warning Buzzer does not Sound).


M14206000911ENG0000010000


cautionWhenever the ECU is replaced, ensure that the input and output signal circuits are normal.


COMMENTS ON TROUBLE SYMPTOM


If the sunroof inhibition function does not work normally, the sunroof motor assembly, the power window main switch or the ETACS-ECU may be defective.

POSSIBLE CAUSES



  • Malfunction of the power window main switch
  • Malfunction of the sunroof motor assembly
  • Malfunction of the ETACS-ECU
  • Damaged harness wires and connectors

DIAGNOSIS PROCEDURE


STEP 1. Which does not operate normally, the sunroof inhibition or the sunroof operation lock buzzer?


Q. Which does not operate normally, the sunroof inhibition or the sunroof operation lock buzzer?

YES (The sunroof inhibition does not work normally.) : Go to Step 2.
YES (The sunroof operation lock buzzer does not work normally.) : Go to Step 4.
YES (Neither operates normally.) : Go to Step 5.
Go to Step 10.

STEP 2. Check for sunroof diagnosis code.


Check whether the sunroof-related diagnosis code is set.

Q. Is the diagnosis code set?

Go to Step 3.
Refer to GROUP54C, Diagnosis Code Chart .

STEP 3. Check the operation of the sunroof.


Check that the sunroof works normally by using the sunroof switch.

Q. Is the check result normal?

The trouble can be an intermittent malfunction (Refer to GROUP 00, How to Use Troubleshooting/Inspection Service Points - How to Cope with Intermittent Malfunction ).
Refer to .

STEP 4. Check the power supply circuit.


When the ignition switch is turned to the LOCK (OFF) position, check if the hazard warning lamps illuminate.

Q. Is the check result normal?

Replace the ETACS-ECU.
Refer to GROUP 54Ad, inspection procedure A-1 "Check the battery power supply circuit to the ETACS-ECU <Not using SWS monitor> or <Using SWS monitor>."

STEP 5. Confirm the operation of the power windows.


Check that each of the power windows (excluding the driver's door) works normally by using the power window main switch.

Q. Is the check result normal?

Go to Step 6.
Refer to GROUP 42B, Trouble Symptom Chart .

STEP 6. Confirm the operation of the power windows.


Turn on the power window lock switch. Check that neither of the power windows (excluding the driver's door) works when the relevant power window sub switch is operated.

Q. Is the check result normal?

Replace to the power window main switch.
Go to Step 7.

STEP 7. Check the operation of the sunroof.


Check that the sunroof works normally by using the sunroof switch.

Q. Is the check result normal?

Go to Step 8.
Refer to .

STEP 8. Check the power supply circuit.


When the ignition switch is turned to the LOCK (OFF) position, check if the hazard warning lamps illuminate.

Q. Is the check result normal?

Go to Step 9.
Refer to GROUP 54Ad, inspection procedure A-1 "Check the battery power supply circuit to the ETACS-ECU <Not using SWS monitor> or <Using SWS monitor>."

STEP 9. M.U.T.-III diagnosis code


When the ignition switch is turned to the LOCK (OFF) position, check that the ETACS-ECU does not set the diagnosis code.

Q. Is the diagnosis code set?

Check that the sunroof inhibition function work normally.
