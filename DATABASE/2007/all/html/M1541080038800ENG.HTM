Code No.U1128 Failures Information on Combination Meter


M15410800388ENG0000010000


cautionIf diagnosis code U1128 is set in the centre display or multi display station, always diagnose the CAN bus lines.

TROUBLE JUDGMENT


The centre display unit or multi display station communicates with the combination meter via CAN bus lines. If failure information is sent to the combination meter, diagnosis code U1128 will be set.

PROBABLE CAUSES



  • Damaged harness wires and connectors
  • Malfunction of combination meter
  • Malfunction of centre display unit or multi display station

DIAGNOSIS PROCEDURE


STEP 1. M.U.T.-III CAN bus diagnostics

Use the M.U.T.-III to diagnose the CAN bus lines.

Q. Is the check result satisfactory?

Go to Step 2.
Repair the CAN bus lines (Refer to GROUP 54D - Troubleshooting ). Repair the CAN bus line and go to Step 6.
STEP 2. M.U.T.-III other system diagnosis code

Check that the combination meter sets a diagnosis code.

Q. Is the diagnosis code set?

Diagnose the combination meter. Refer to GROUP 54Ab - Troubleshooting .
Go to Step 3.
STEP 3. Check the combination meter operation.

Check that the fuel gauge in the combination meter works normally.

Q. Is the check result satisfactory?

Go to Step 4.
Diagnose the combination meter. Refer to GROUP 54Ab - Troubleshooting .
STEP 4. Check whether the diagnosis code is reset.

Replace the centre display unit or multi display station temporarily, and then check that the diagnosis code is not reset.
(1)Replace the centre display unit or multi display station temporarily.
(2)Erase the diagnosis code.
(3)Turn the ignition Switch to "ON" position.
(4)Check if the diagnosis code is set.

Q. Is the check result satisfactory?

Replace centre display unit or multi display station and then go to step 6.
Go to Step 5.
STEP 5. Check whether the diagnosis code is reset.

Replace the combination meter temporarily, and then check that the diagnosis code is not reset.
(1)Replace the combination meter temporarily.
(2)Erase the diagnosis code.
(3)Turn the ignition Switch to "ON" position.
(4)Check if the diagnosis code is set.

Q. Is the check result satisfactory?

A poor connection, open circuit or other intermittent malfunction is present in the lines between the combination meter and the centre display unit or multi display station (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points - How to Cope with Intermittent Malfunctions ).
Replace combination meter and then go to step 6.
STEP 6. Check whether the diagnosis code is reset.

Recheck if the diagnosis code is set.
(1)Erase the diagnosis code.
(2)Turn the ignition Switch to "ON" position.
(3)Check if the diagnosis code is set.

Q. Is the check result satisfactory?

The procedure is complete.
Go to Step 1.
