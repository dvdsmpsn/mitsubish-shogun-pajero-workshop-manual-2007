A coil spring type double wishbone independent type suspension has been adopted as the front suspension. The suspension which is the same as the conventional PAJERO has been changed as follows to improve handling stability and riding comfort.

  • The wheel tread has been widened by 10 mm for cornering ability. <Long wheel base>
  • The spring constant has been changed for handling stability and riding comfort.
  • The damping force has been changed for handling stability and riding comfort.
