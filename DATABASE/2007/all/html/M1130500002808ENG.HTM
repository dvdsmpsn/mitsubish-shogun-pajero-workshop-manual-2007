REMOVAL SERVICE POINT


<<A>> FUEL FILTER CARTRIDGE REMOVAL



Fix the fuel filter pump to a vice, and then remove the fuel filter cartridge using an oil filter wrench.

INSTALLATION SERVICE POINTS


>>A<< FUEL FILTER CARTRIDGE INSTALLATION


1. Clean the mounting surface on the fuel filter pump side.
2. Tighten the fuel filter cartridge to the specified torque from the point where the gasket contacts the mounting surface.
Tightening torque: Approximately 3/4 - 1 turn (17 ± 2 N·m)

>>B<< FUEL HOSE INSTALLATION



If the pipe has a stepped part, connect securely up to the stepped part. If the pipe has no stepped part, insert so that the inserted portion 20 - 25 mm long.
