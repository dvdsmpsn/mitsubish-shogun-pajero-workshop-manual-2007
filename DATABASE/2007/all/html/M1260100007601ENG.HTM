EUJ BOOT REPLACEMENT


M12601000076ENG0000010000



1. Remove the boot bands (large and small).
note The EUJ boot bands cannot be re-used.
2. Remove the EUJ boot.

3. Install the small part on the boot aligning the groove of the shaft.

4. Turn the adjusting bolt on special tool boot band clipping tool (MB991561) so that the size of the opening (W) is at the standard value.
Standard value (W): 2.9 mm

    <If it is smaller than 2.9 mm> loosen the adjusting bolt.
    <If it is larger than 2.9 mm> tighten the adjusting bolt.
noteThe value of W will change by approximately 0.7 mm for each turn of the adjusting bolt.
noteThe adjusting bolt should not be turned more than once.

5. Install the EUJ boot band (small) to the EUJ boot.

6. Use special tool boot band clipping tool (MB991561) to crimp the EUJ boot band (small).
caution
  • Secure the drive shaft in an upright position and clamp the part of the EUJ boot band to be crimped securely in the jaws of special tool MB991561.

  • Crimp the EUJ boot band until the special tool touches the stopper.

7. Check that crimping amount (A) of the EUJ boot band is at the standard value.
Standard value (A): 2.4 - 2.8 mm

    <If the crimping amount is larger than 2.8 mm>
    Readjust the value of (W) in step 4 according to the following formula, and then repeat the operation in step 6.
    W = 5.5 mm - A
    Example: If A = 2.9 mm, then W = 2.6 mm.
    <If the crimping amount is smaller than 2.4 mm
    Remove the EUJ boot band, readjust the value of (W) in step 4 according to the following formula, and then repeat the operations in steps 5 and 6 using a new BJ boot band.
    W = 5.5 mm - A
    Example: If A = 2.3 mm then W = 3.2 mm.
8. Check that the EUJ boot band is not sticking out past the place where it has been installed. If the EUJ boot band is sticking out, remove it and then repeat the operations in steps 4 to 7 using a new EUJ boot band.
9. Fill the inside of the EUJ boot with the specified amount of the specified grease.

    Specified grease: Repair kit grease
    Amount to use: 175 ± 10 g

10. Install the large diameter part on the plastic boot aligning the groove of the shaft.
11. Follow the same procedure as in step 4 to adjust the size of the opening (W) on special tool MB991561 so that it is at the standard value.
Standard value (W): 3.2 mm

12. Install the EUJ boot band (large) to the EUJ boot.
13. Use special tool MB991561 to crimp the EUJ boot band (large) in the same way as in step 6.

14. Check that the crimping amount (B) of the EUJ boot band is at the standard value and that the crimping amount (C) of the EUJ boot band is at the limited value.

    Standard value (B): 2.4 - 2.8 mm

      <If the crimping amount is larger than 2.8 mm>
      Readjust the value of (W) in step 11 according to the following formula, and then repeat the operation in step 13.
      W = 5.8 mm - B
      Example: If B = 2.9 mm, then W = 2.9 mm.
      <If the crimping amount is smaller than 2.4 mm>
      Remove the EUJ boot band, readjust the value of (W) in step 11 according to the following formula, and then repeat the operations in steps 12 and 13 using a new EUJ boot band.
      W = 5.8 mm - B
      Example: If B = 2.3 mm then W = 3.5 mm.
15. Check that the EUJ boot band is not sticking out past the place where it has been installed. If the EUJ boot band is sticking out, remove it and then repeat the operations in steps 12 to 14 using a new EUJ boot band.
