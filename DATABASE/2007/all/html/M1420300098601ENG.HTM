INSPECTION PROCEDURE 4: The Key Reminder Function does not work Normally. <Not using SWS monitor>


M14203000986ENG0000010000


cautionWhenever the ECU is replaced, ensure that the input and output signal circuits are normal.


COMMENTS ON TROUBLE SYMPTOM


If the key reminder function does not work normally, the input signal circuits below or the ETACS-ECU may be defective.

  • Key reminder switch
  • Driver's door switch
  • Driver's door lock actuator

POSSIBLE CAUSES



  • Malfunction of the key reminder switch
  • Malfunction of the driver's door switch
  • Malfunction of the driver's door lock actuator
  • Malfunction of the ETACS-ECU
  • Damaged harness wires and connectors

DIAGNOSIS PROCEDURE


STEP 1. Check the power supply circuit.


When the ignition switch is turned to the LOCK (OFF) position, check if the hazard warning lamps illuminate.

Q. Is the check result normal?

Go to Step 2.
Refer to GROUP 54Ad, inspection procedure A-1 "Check the battery power supply circuit to the ETACS-ECU <Not using SWS monitor>."

STEP 2. Pulse check


Check the input signals belows which are related to the key reminder function.
System switch
Check condition
Driver's door switch
When the driver's door is opened
Key reminder switch
When the inserted ignition key is pulled out
Driver's door lock actuator switch
When the driver's key cylinder or inside lock knob is unlocked or locked
OK: The M.U.T.-III sounds or the voltmeter needle fluctuates.

Q. Are the check result normal?

YES (All the signals are received normally.) : Go to Step 3.
NO (The driver's door switch signal is not received.) : Refer to GROUP 54Ad, Inspection Procedure C-5 "The driver's door switch signal is not received LH drive vehicles or RH drive vehicles ."
NO (The key reminder switch signal is not received.) : Refer to GROUP 54Ad Inspection Procedure C-11 "The key reminder switch signal is not received ."
NO (The driver's door lock actuator switch signal is not received.) : Refer to GROUP 54Ad, inspection procedure C-15 "The driver's door lock actuator switch signal is not received LH drive vehicles or RH drive vehicles ."

STEP 3. Retest the system.


Q. Does the key reminder function work normally?

The trouble can be an intermittent malfunction (Refer to GROUP 00, How to Use Troubleshooting/Inspection Service Points - How to Cope with Intermittent Malfunction ).
Replace the ETACS-ECU.
