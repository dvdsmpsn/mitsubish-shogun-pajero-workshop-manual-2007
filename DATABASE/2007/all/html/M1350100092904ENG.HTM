Code No.55: Pump motor system (motor energized too long)


M13501000929ENG0000010001




OPERATION



  • The pump motor operates if the motor relay 1 or motor relay 2 turn to ON.
  • The ASTC-ECU monitors the pump motor (at 25 terminal).

DIAGNOSIS CODE SET CONDITIONS


Diagnosis code No.55 will be set when the pump motor has been energized too long. The ASTC-ECU sounds the buzzer continuously to inform the trouble of the driver and protects the pump motor by activating the pump motor intermittently.

PROBABLE CAUSES



  • Malfunction of the motor relay 1 or the motor relay 2
  • Malfunction of the pressure switch (for pump motor control) [Integrated with the hydraulic brake booster (HBB)]
  • Damaged wiring harness or connector
  • Malfunction of the ASTC-ECU
