Code No.U1100 Engine-ECU <M/T> or Engine-A/T-ECU <A/T> Time-out (Related to Engine)


M15410800384ENG0000010000


caution
  • If diagnosis code U1100 is set in the centre display or multi display station, always diagnose the CAN bus lines.
  • Whenever the ECU is replaced, ensure that the communication circuit is normal.

TROUBLE JUDGMENT


The centre display unit or multi display station receives engine control system-related signals from the engine-ECU <M/T> or engine-A/T-ECU <A/T> via CAN bus lines. If the centre display unit or multi display station can not receives the necessary signals, diagnosis code U1100 will be set.

PROBABLE CAUSES



  • Damaged harness wires and connectors
  • Malfunction of engine-ECU <M/T> or engine-A/T-ECU <A/T>
  • Malfunction of centre display unit or multi display station

DIAGNOSIS PROCEDURE


STEP 1. M.U.T.-III CAN bus diagnostics

Use the M.U.T.-III to diagnose the CAN bus lines.

Q. Is the check result satisfactory?

Go to Step 2.
Repair the CAN bus lines (Refer to GROUP 54D - Troubleshooting ). Repair the CAN bus line and go to Step 6.
STEP 2. M.U.T.-III other system diagnosis code

Check if an MPI system <3800> or diesel fuel system <3200> diagnosis code is set.

Q. Is the diagnosis code set?

Diagnose the MPI system <3800> or diesel fuel system <3200> (Refer to GROUP 13Ab - Troubleshooting <3800> or GROUP 13Bb - Troubleshooting ).
Go to Step 3.
STEP 3. M.U.T.-III other system diagnosis code

Check if a diagnosis code, which relates to CAN communication-linked systems below, is set.

  • ASC-ECU

      • Code No.U1100: engine-ECU <M/T> or engine-A/T-ECU <A/T> time-out (related to Engine)
  • ETACS-ECU

      • Code No.U1100: engine-ECU <M/T> or engine-A/T-ECU <A/T> time-out (related to Engine)
  • A/C-ECU

      • Code No.U1100: engine-ECU <M/T> or engine-A/T-ECU <A/T> time-out (related to Engine)

Q. Is the diagnosis code set?

Go to Step 4.
Go to Step 5.
STEP 4. Check whether the diagnosis code is reset.

Recheck if the diagnosis code is set.
(1)Erase the diagnosis code.
(2)Turn the ignition switch to "ON" position.
(3)Check if the diagnosis code is set.

Q. Is the check result satisfactory?

A poor connection, open circuit or other intermittent malfunction is present in the CAN bus lines between the engine-ECU <M/T> or engine-A/T-ECU <A/T> and the centre display unit or multi display station (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points - How to Cope with Intermittent Malfunctions ).
Replace engine-ECU <M/T> or engine-A/T-ECU <A/T> and then go to step 6.
STEP 5. Check whether the diagnosis code is reset.

Recheck if the diagnosis code is set.
(1)Erase the diagnosis code.
(2)Turn the ignition switch to "ON" position.
(3)Check if the diagnosis code is set.

Q. Is the check result satisfactory?

A poor connection, open circuit or other intermittent malfunction is present in the CAN bus lines between the engine-ECU <M/T> or engine-A/T-ECU <A/T> and the centre display unit or multi display station (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points - How to Cope with Intermittent Malfunctions ).
Replace centre display unit or multi display station and then go to step 6.
STEP 6. Check whether the diagnosis code is reset.

Recheck if the diagnosis code is set.
(1)Erase the diagnosis code.
(2)Turn the ignition switch to "ON" position.
(3)Check if the diagnosis code is set.

Q. Is the check result satisfactory?

The procedure is complete.
Go to Step 1.
