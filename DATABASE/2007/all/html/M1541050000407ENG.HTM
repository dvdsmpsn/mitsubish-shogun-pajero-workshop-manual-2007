PRINTED HEATER CHECK


M15410500004ENG0000010000



1. Run the engine at a speed of 2,000 r/min to fully charge the battery, and then check the printed heater.
2. With the defogger switch turned on, use a multimeter to measure the voltages of each printed heater at centre point A on the window glass. The printed heater is okay if 6 V is displayed.
3. If a voltage of 12 V is indicated at point A, the open circuit location is between point A and the (-) terminal to find the point where the voltage changes is the location of the open circuit.
4. In addition, if a voltage of 0 V is indicated at point A, the open circuit location is between point A and the (+) terminal. Use the same procedure as described above to find the point where the voltage changed to 12 V.
