PTC HEATER CHECK


Check that there is continuity between the connector terminals 1 and 2.

REAR BLOWER MOTOR CHECK


Check that the motor turns when applying battery power between the connector terminals. Also check to see that there is no abnormal sound emitted from the motor at this time.

RESISTOR CHECK


Use an ohmmeter to measure the resistance between the terminals. Check that the measured value is at the standard value.
Standard value:
Measurement terminal
Standard value (Ω)
