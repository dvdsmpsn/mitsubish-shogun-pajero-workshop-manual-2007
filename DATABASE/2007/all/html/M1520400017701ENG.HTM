Code No.B1426: G-sensor of side impact sensor (RH) failure
Code No.B1436: G-sensor of side impact sensor (LH) failure


M15204000177ENG0000010000


cautionIf diagnosis code B1426 or B1436 is set in the SRS-ECU, always diagnose the CAN main bus line.

DIAGNOSIS CODE SET CONDITIONS


These diagnosis codes are set if the following conditions are detected from the analogue G-sensor inside the side impact sensor output:

  • Analogue G-sensor is not operating.

  • Analogue G-sensor characteristics are abnormal.

  • Analogue G-sensor output is abnormal.

PROBABLE CAUSES


Malfunction of side impact sensor (front) (RH) (for diagnosis code B1426) and side impact sensor (front) (LH) (for diagnosis code B1436)

DIAGNOSIS PROCEDURE


STEP 1. M.U.T.-III CAN bus diagnostics


Use the M.U.T.-III to diagnose the CAN bus lines.

Q. Is the check result normal?

Go to Step 2.
Repair the CAN bus line (Refer to GROUP 54D, Diagnosis ).

STEP 2. Check whether the diagnosis code is reset.


Check again if the diagnosis code is set.
(1)Erase the diagnosis code.
(2)Ignition: "LOCK" (OFF) position to "ON"
(3)On completion, check that the diagnosis code is not reset.

Q. Is the diagnosis code set?

Go to Step 3.
There is an intermittent malfunction such as poor engaged connector(s) or open circuit (Refer to GROUP 00, How to Cope with Intermittent Malfunction ).

STEP 3. Check the side impact sensor (front) (M.U.T.-III diagnosis code).


Replace side impact sensor (front) (RH) (for diagnosis code B1426) and side impact sensor (front) (LH) (for diagnosis code B1436) (Refer to ).
Check the diagnosis code.

Q. Is either diagnosis code B1426 or B1436 set?

Replace the SRS-ECU (Refer to ).
The procedure is complete.
