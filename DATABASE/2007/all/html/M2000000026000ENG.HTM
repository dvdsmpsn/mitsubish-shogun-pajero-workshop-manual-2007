IMMOBILIZER SYSTEM


<3200>

<3800>

The immobilizer system consists of the ignition key, the key ring antenna, the immobilizer-ECU and the engine-ECU <3200>, engine-A/T-ECU <3800>.The igniting key has a built-in transponder as the oscillator. The key ring antenna is installed on the steering lock key cylinder. Only the registered ignition key permits the engine to start, therefore, the engine can never be started by means of a forged key or by connecting the ignition wiring directly. The system is significantly safe and reliable against theft. In addition, the driver has only to turn the ignition switch to the "ON" position to activate the immobilizer system.If the ignition key is lost or another ignition key is added, all the key must be registered again by using the scan tool MB991502 (M.U.T.-III) for security reasons.

AUDIO AND NAVIGATION SYSTEM


RADIO AND CD PLAYER




An unique panel audio system (6-disk CD changer with AM/FM/LW/MW radio, or CD player with AM/FM/LW/MW radio) imparts a uniformity with the instrument panel. This system adjusts sound volume and quality automatically according to the vehicle driving conditions.

MITUBISHI MULTI COMMUNICATION SYSTEM (MMCS) <OPTION>




The Mitsubishi multi-communication system (HDD navigation system with 7-inch wide monitor) incorporates a 30-GB hard disk. Moreover, the wide 2-DIN size 7-inch LCD display is adopted.

AUDIO REMOTE CNTROL SWITCH <GLS>




A centralised switch is located on the steering wheel spoke to allow you to control the audio system remotely.
