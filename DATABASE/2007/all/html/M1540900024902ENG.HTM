Code No.05 Communication Error with Power Window Main Switch


M15409000249ENG0000010001


cautionIf diagnosis code No.05 is set in the ETACS-ECU, diagnose the CAN bus lines.

TROUBLE JUDGMENT


The ETACS-ECU communicates with the power window main switch through the SWS communication lines. If there is any trouble in that communication, diagnosis code No.05 will be set.

COMMENTS ON TROUBLE SYMPTOM


If the power window main switch, the ETACS-ECU or the CAN bus line fails, this diagnosis code will be set.

PROBABLE CAUSES



  • Malfunction of the power window main switch
  • Malfunction of the ETACS-ECU
  • Malfunction of the CAN bus

DIAGNOSIS PROCEDURE


STEP 1. M.U.T.-III CAN bus diagnostics


Use the M.U.T.-III to diagnose the CAN bus lines.

Q. Is the check result normal?

Go to Step 2.
Repair the CAN bus line (Refer to GROUP 54D - Troubleshooting ).

STEP 2. Check whether the diagnosis code is reset.


Check again if the diagnosis code is set.
(1)Erase the diagnosis code.
(2)Ignition switch: LOCK (OFF) position to ON
(3)On completion, check that the diagnosis code is not reset.

Q. Is the diagnosis code set?

Go to Step 3.
Intermittent malfunction (Refer to GROUP 00 - How to use troubleshooting/Inspection Service Points ).

STEP 3. ECU check by using the SWS monitor


Check that the power supply and earth lines to the ETACS-ECU and the power window main switch (power window module) and the SWS communication lines are normal.

  • Ignition switch: ON
ECUS TO BE CHECKED
  • P/W MODULE
  • ETACS ECU
OK: "OK" are displayed for all the items

Q. Are the check result normal?

YES <Normal conditions are displayed for all the items.> : Intermittent malfunction (Refer to GROUP 00 - How to use troubleshooting/Inspection Service Points ).
NO <"NG" is displayed on the "ETACS ECU" menu.> : Refer to Inspection Procedure 2 "Communication with the ETACS-ECU is not possible ."
NO <"NG" is displayed on the "P/W MODULE" menu.> : Refer to Inspection Procedure 4 "Communication with the power window main switch (power window module) is not possible ."
