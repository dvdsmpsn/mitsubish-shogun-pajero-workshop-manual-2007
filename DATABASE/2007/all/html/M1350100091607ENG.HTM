BRAKE WARNING LAMP INSPECTION


M13501000916ENG0000010000



Check that the brake warning lamp illuminates as follows.

  1. When the ignition key is turned to ’’ON’’, the brake warning lamp illuminates for approximately 3 seconds and then switches off.
  2. When the ignition key is turned to ’’START’’, the brake warning lamp remains illuminated.
  3. When the ignition key is turned from ’’START’’ back to ’’ON’’, the brake warning lamp illuminates for approximately 3 seconds and then switches off.
  4. If the illumination is other than the above, check the diagnosis codes.
