caution
  • Never loosen the screw fixing the accelerator pedal assembly resin cover. If the screw is loosened, the sensor position which exists inside the resin cover is off and the accelerator pedal position sensor does not work normally.
  • Do not remove the accelerator pedal pad. If the pad is removed and installed, excessive force may damage accelerator pedal position sensor.
