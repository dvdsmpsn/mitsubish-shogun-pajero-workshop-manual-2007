DISASSEMBLY SERVICE POINTS


<<A>> ROLLER BEARING REMOVAL



Using special tool Bearing remover (MD998801), remove the roller bearing to the counter shaft cluster gear.

<<B>> BEARING SLEEVE REMOVAL



Using special tool Bearing remover (MD998801), remove the bearing sleeve to the counter shaft cluster gear.

<<C>> ROLLER BEARING REMOVAL



Using special tool Bearing remover (MD998801), remove the roller bearing to the counter shaft cluster gear.

REASSEMBLY SERVICE POINTS


>>A<< ROLLER BEARING INSTALLATION



Using special tools to install the roller bearing to the counter shaft cluster gear.

  • Installer cup (MD998812)
  • Installer-200 (MD998814)
  • Installer adaptor (40) (MD998819)

>>B<< BEARING SLEEVE INSTALLATION



Using special tools to install the bearing sleeve to the counter shaft cluster gear.

  • Installer cup (MD998812)
  • Installer-100 (MD998813)
  • Installer adaptor (54) (MD998826)

>>C<< ROLLER BEARING INSTALLATION



Set special tool Bearing remover (MD998801), using special tools to install the roller bearing to the counter shaft cluster gear.

  • Installer cup (MD998812)
  • Installer-100 (MD998813)
  • Installer adaptor (38) (MD998818)

>>D<< SNAP RING INSTALLATION



Select a snap ring which will make the counter shaft cluster gear bearing end play equal to the standard value.
Standard value: 0 - 0.15 mm
