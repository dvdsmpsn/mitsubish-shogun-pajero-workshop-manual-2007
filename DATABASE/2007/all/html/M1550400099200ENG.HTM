Code No. B1093: Photo sensor system


M15504000992ENG0000010000




COMMENTS ON TROUBLE SYMPTOM


This code is set when the photo sensor circuit is short.

PROBABLE CAUSES



  • Malfunction of the photo sensor
  • Damaged the wiring harness or connectors
  • Malfunction of the automatic A/C control panel (A/C-ECU)

DIAGNOSIS PROCEDURE


STEP 1. Connector check: D-08 A/C-ECU connector and D-06-1 photo sensor connector






Q. Is the check result normal?

Go to Step 2.
Repair the connector.

STEP 2. Check the wiring harness between D-08 A/C-ECU connector (terminals 26 and 8) and D-06-1 photo sensor connector (terminals 5 and 4).






notePrior to the wiring harness inspection, check intermediate connectors D-06, and repair if necessary.

  • Check the sensor signal line and earth line for open or short circuit.

Q. Is the check result normal?

Go to Step 3.
Repair the wiring harness.

STEP 3. Check the photo sensor.


Refer to .

Q. Is the check result normal?

Go to Step 4.
Replace the photo sensor.

Q. Is the diagnosis code set?
