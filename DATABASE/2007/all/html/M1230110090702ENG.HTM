Code No.P1770 (P0705): Inhibitor switch system


M12301100907ENG0000010000




OPERATION


The inhibitor switch detects the selector lever position (P, R, N,D) which the driver has selected, and sends the information to the engine-A/T-ECU.

DIAGNOSIS CODE SET CONDITIONS


If the inhibitor switch has not been sending any signal for at least 30 seconds, an open circuit may be present and diagnosis code No.P1770 (P0705) is set.
PROBABLE CAUSES
  • Malfunction of the inhibitor switch
  • Damaged harness wires and connectors
  • Malfunction of the engine-A/T-ECU
