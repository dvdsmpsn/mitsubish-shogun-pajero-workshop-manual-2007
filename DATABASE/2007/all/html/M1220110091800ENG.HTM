Code No.11, 12: Power supply voltage system


M12201100918ENG0000010000




OPERATION


When turning the ignition switch to ON, the transfer-ECU (terminal No.13) receives system voltage from the ignition switch.

DIAGNOSIS CODE SET CONDITIONS


When power supply voltage drops to 9.5 V and less, the code No.11 is set as low voltage.
When power supply voltage rises to 18 V and over, the code No.12 is set as high voltage.

PROBABLE CAUSES



  • Malfunction of the ignition switch
  • Damaged harness wires and connectors
  • Malfunction of the transfer-ECU
