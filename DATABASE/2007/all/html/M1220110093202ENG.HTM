Code No.45: Dissimilar tyre diameter


M12201100932ENG0000010000


OPERATION


The condition of tyre is detected by monitoring the speed difference between the front propeller shaft speed sensor and the rear propeller shaft speed sensor with the transfer-ECU at 4WD condition.

DIAGNOSIS CODE SET CONDITIONS


If the speed difference between the front wheel and the rear wheel is larger than the set value at the freewheel engage switch ON condition, the code No.45 is set as the inadequate tyre pressure, non-conformity of the tyre size or the tyre brand.

PROBABLE CAUSES



  • Malfunction of the tyre
  • Malfunction of the transfer-ECU

DIAGNOSIS PROCEDURE


STEP 1. Tyre pressure check.


Check the tyre pressure, size and brand.

Q. Is the tyre condition normal?

Go to Step 2.
Adjust or replace the tyre.

STEP 2. M.U.T.-III diagnosis code.


(1)Test drive the vehicle.
(2)Check if the diagnosis code is set.

Q. Is diagnosis code 45 set?

Replace the transfer-ECU.
Intermittent malfunction (Refer to GROUP 00 - How to Cope with Intermittent Malfunction ).
