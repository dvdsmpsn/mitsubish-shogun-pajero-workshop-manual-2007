REMOVAL SERVICE POINTS


<<A>> ROCKER ARM AND SHAFT ASSEMBLY REMOVAL



1. Before rocker arm and shaft assembly removal, install special tool lash adjuster holder (MD998443) as shown in the illustration so that the rocker arm lash adjusters will not fall out.
2.
cautionNever disassemble the rocker arm, shaft and rocker arm lash adjuster assembly.
Loosen the rocker arm and shaft assembly mounting bolt, and then remove the rocker arm and shaft assembly with the bolt still attached.

<<B>> VALVE SPRING RETAINER LOCK REMOVAL


cautionWhen removing valve spring retainer lock, leave the piston of each cylinder in the TDC (Top Dead Centre) position. The valve may fall into the cylinder if the piston is not properly in the TDC position.


Use special tool valve spring compressor (MD998772) to compress the valve spring, and remove the valve spring retainer locks.

INSTALLATION SERVICE POINTS


>>A<< VALVE STEM SEAL INSTALLATION


1. Apply a small amount of engine oil to the valve stem seal.

2.
caution
  • Valve stem seals cannot be reused.
  • The special tool valve stem seal installer (MB991999) must be used to install the valve stem seal. Improper installation could result in oil leaking past the valve guide.
Use special tool (MB991999) to fill a new valve stem seal in the valve guide using the valve stem area as a guide.

>>B<< VALVE SPRING INSTALLATION



Install the valve springs with its identification colour painted end facing the locker arm.

>>C<< VALVE SPRING RETAINER LOCK INSTALLATION




Use special tool valve spring compressor (MD998772) to compress the valve spring in the same manner as removal, install the valve spring retainer lock.

>>D<< ROCKER ARM AND SHAFT ASSEMBLY INSTALLATION


1. Ensure that all rocker arms do not press against the valve, then pre-tighten the rocker arm and shaft assembly with the mounting bolt.

2. Check that notches in the each rocker arm shaft are facing the direction shown in the illustration.
3. Tighten the rocker arm and shaft assembly mounting bolts to the specified torque.

    Tightening torque:

      <Inlet side> 31 ± 3 N·m
      <Exhaust side> 12 ± 2 N·m

4. Remove the special tool lash adjuster holder (MD998443) which was installed to the rocker arm when removing the rocker arm and shaft assembly.
