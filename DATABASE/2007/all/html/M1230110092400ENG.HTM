Code No.U1190: Can’t receive fault detection control signal


M12301100924ENG0000010000


DIAGNOSIS CODE SET CONDITIONS


To each ECU connected to the CAN bus line, ETACS-ECU sends the diagnosis code (U code) detection permission or prohibition signal. When engine-A/T-ECU cannot receive the diagnosis code (U code) detection permission or prohibition signal for 5 seconds, diagnosis code No. U1190 is stored.

PROBABLE CAUSES



  • Malfunction of the ETACS-ECU
  • Malfunction of the CAN bus

DIAGNOSIS PROCEDURE


STEP 1. M.U.T.-III CAN bus diagnostics


Using M.U.T.-III, perform CAN busline diagnosis.

Q. Is the check result normal?

Go to Step 2.
Repair the CAN busline (Refer to GROUP 54D - Troubleshooting ). and then go to Step 3.

STEP 2. M.U.T.-III diagnosis code


Reconfirm whether the diagnosis codes are output from engine-A/T-ECU.
(1)Erase the diagnosis codes being output.
(2)Confirm whether the diagnosis codes are output.

Q. Is the diagnosis code output?

Replace the ETACS-ECU, and then go to Step 3.
The procedure is complete.

STEP 3. M.U.T.-III diagnosis code


Reconfirm whether the diagnosis codes are output from engine-A/T-ECU.
(1)Erase the diagnosis codes being output.
(2)Confirm whether the diagnosis codes are output.

Q. Is the diagnosis code output?

Return to Step 1.
The procedure is complete.
