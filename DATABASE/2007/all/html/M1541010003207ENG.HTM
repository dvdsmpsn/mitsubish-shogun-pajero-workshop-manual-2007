Inspection Procedure 4: Fuel Gauge does not work (other meters work).


M15410100032ENG0000010000


COMMENTS ON TROUBLE SYMPTOM


If the fuel gauge does not operate normally, DTC B1201 may have been stored with the combination meter.

PROBABLE CAUSES



  • Damaged harness wires and connectors
  • Malfunction of combination meter

DIAGNOSIS PROCEDURE


M.U.T.-III diagnosis code.


Check that a combination meter DTC has been stored.
