REMOVAL SERVICE POINTS


<<A>> POWER WINDOW SWITCH PANEL ASSEMBLY REMOVAL



There is a clip (driver's side) or a claw (other sides) on the front of the power window switch assembly. Work from the front side using special tool ornament remover (MB990784).

<<B>> DOOR TRIM REMOVAL



1. Insert the special tool ornament remover (MB990784) between the inside handle upper part and the inside handle cover, and then disengage the upper claw of the inside handle.
2. Disengage the lower claw of the inside handle in the same manner as for the upper claw.
3. Remove the door trim.
4. Remove the inside handle cover from the door trim.
