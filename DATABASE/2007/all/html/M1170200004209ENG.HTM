Inspection Procedure 4: Auto-cruise Control cannot be Set <4M4>.


M11702000042ENG0000010003


COMMENTS ON TROUBLE SYMPTOM


The fail-safe function is probably cancelling auto-cruise control. In this case, checking the auto-cruise control system and diesel fuel system diagnosis codes. The M.U.T.-III can also be used to check if the circuits of each input switch are normal or not by checking the input switch codes.

PROBABLE CAUSES



  • Malfunction of the diesel fuel system.
  • Malfunction of the auto-cruise control switch.
  • Malfunction of the stop lamp switch.
  • Malfunction of the clutch switch.
  • Malfunction of the engine-ECU.

DIAGNOSIS PROCEDURE


STEP 1. Check the diesel fuel system diagnosis code.


Refer to GROUP 13Bb - Diagnosis Function .

Q. Is any diagnosis code set?

Repair the diesel fuel system. (Refer to GROUP 13Bb - Inspection Chart for Diagnosis Code ). Then go to Step 5.
Go to Step 2.

Read the auto-cruise control system diagnosis code (Refer to ).

Diagnose the auto-cruise control system [Refer to , Check Chart for Diagnosis Codes <4M4>]. Then go to Step 5.
Go to Step 3.

STEP 3. M.U.T.-III data list.


(1)

Go to Step 4

STEP 4. Check the symptoms.


Q. Can auto-cruise control be set?

It can be assumed that this malfunction is intermittent (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunctions ).
Replace the engine-ECU (Refer to GROUP 13Ba - Engine-ECU ). Then go to Step 5.

STEP 5. Check the symptoms.


Q. Can auto-cruise control be set?

The procedure is complete.
