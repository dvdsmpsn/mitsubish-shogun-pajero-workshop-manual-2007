Code No.U1108 Combination Meter Time-out


M15410800385ENG0000010000


caution
  • If diagnosis code U1108 is set in the centre display or multi display station, always diagnose the CAN bus lines.
  • Whenever the ECU is replaced, ensure that the communication circuit is normal.

TROUBLE JUDGMENT


The centre display unit or multi display station receives signals from the combination meter via CAN bus lines. If the centre display unit or multi display station can not receive the necessary combination meter signals, diagnosis code U1108 will be set.

PROBABLE CAUSES



  • Damaged harness wires and connectors
  • Malfunction of combination meter
  • Malfunction of centre display unit or multi display station

DIAGNOSIS PROCEDURE


STEP 1. M.U.T.-III CAN bus diagnostics

Use the M.U.T.-III to diagnose the CAN bus lines.

Q. Is the check result satisfactory?

Go to Step 2.
Repair the CAN bus lines (Refer to GROUP 54D - Troubleshooting ). Repair the CAN bus line and go to Step 6.
STEP 2. M.U.T.-III other system diagnosis code

Check whether the combination meter-related diagnosis code is set.

Q. Is the diagnosis code set?

Diagnose the combination meter. (Refer to GROUP 54Ab - Troubleshooting .)
Go to Step 3.
STEP 3. M.U.T.-III other system diagnosis code

Check if a diagnosis code, which relates to CAN communication-linked systems below, is set.

  • MPI

      • Code No.U1108: Combination meter time-out
  • ETACS-ECU

      • Code No.14: Combination meter time-out

Q. Is the diagnosis code set?

Go to Step 4.
Go to Step 5.
STEP 4. Check whether the diagnosis code is reset.

Recheck if the diagnosis code is set.
(1)Erase the diagnosis code.
(2)Turn the ignition switch to "ON" position.
(3)Check if the diagnosis code is set.

Q. Is the check result satisfactory?

A poor connection, open circuit or other intermittent malfunction is present in the CAN bus lines between the combination meter and the centre display unit or multi display station (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points - How to Cope with Intermittent Malfunctions ).
Replace combination meter and then go to step 6.
STEP 5. Check whether the diagnosis code is reset.

Recheck if the diagnosis code is set.
(1)Erase the diagnosis code.
(2)Turn the ignition switch to "ON" position.
(3)Check if the diagnosis code is set.

Q. Is the check result satisfactory?

A poor connection, open circuit or other intermittent malfunction is present in the CAN bus lines between the combination meter and the centre display unit or multi display station (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points - How to Cope with Intermittent Malfunctions ).
Replace centre display unit or multi display station and then go to step 6.
STEP 6. Check whether the diagnosis code is reset.

The procedure is complete.
