MAINTENANCE-FREE FEATURES


Serpentine auxiliary belts and an auto-tensioner are adopted to eliminate the need for the adjustment of auxiliary belt tension.

MULTI USE TESTER (M.U.T.-III)


Comprehensive improvements have been made to the M.U.T.-II, a tester for diagnosing problems with the electronic control system. For easier servicing, the newly developed M.U.T.-III has greatly improved functions and is much easier to use. The M.U.T.-III expands the functions of the M.U.T.-II in the following ways:

  1. Interactive Error Diagnosis

    • In response to the nature of the problem, the corresponding troubleshooting page from the workshop manual is retrieved.
    • Service data is displayed, and from the actuator test screen, the page of the workshop manual is retrieved for a list of inspection reference values.
  2. Service Manual Viewer

    • The technical information manual and workshop manual can be displayed on a personal computer monitor.
  3. CAN* bus diagnosis

    • Auto diagnosis function for the CAN communications bus line is added.
note*CAN: Controller Area Network (for further details, refer to GROUP 54C ).

ENHANCED DIAGNOSIS SYSTEM


Diagnosis functions have been included for the following systems, so that it is possible to use the M.U.T.-III to read the diagnosis codes and service data and to carry out actuator tests. In addition, it is also possible to read the diagnosis codes by the flashing of the warning lamp in some systems.

  • Multi-Point Injection system (MPI)
  • Auto-cruise control
  • 5A/T
  • Super select 4WD II
  • ABS (with EBD)
  • Mitsubihi Active Stability Traction Control (M-ASTC)
  • Power windows
  • SRS air bag
  • Immobilizer
  • Combination meter
  • Headlamp auto levelling
  • ETACS
  • Heater, air conditioner, and ventilation

IMPROVED SERVICEABILITY AND HANDLING


The serpentine system with the auto-tensioner is adopted for the accessory belt to realize maintenance free and to improve the ease of replacement work.
