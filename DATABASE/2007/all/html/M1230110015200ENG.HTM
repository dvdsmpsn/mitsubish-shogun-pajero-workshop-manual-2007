Code No.21: Accelerator pedal position sensor system


M12301100152ENG0000010000




OPERATION



  • When the accelerator pedal is from the idle position to the fully opened position, the resistance between the accelerator pedal position sensor output terminal (No.6) and earth terminal (No.5) will increase according.
  • With the ignition switch in the ON position. Voltage at terminal No.6 increases from approximately 0.5 V to approximately 2.325 V.

DIAGNOSIS CODE SET CONDITIONS


If accelerator pedal position sensor output voltage is 0.2 V or lower at times other than when the engine is idling, the output is judged to be too low and code No.21 is set.

PROBABLE CAUSES



  • Malfunction of the accelerator pedal position sensor circuit
  • Damaged harness wires and connectors
  • Malfunction of the transfer-ECU
