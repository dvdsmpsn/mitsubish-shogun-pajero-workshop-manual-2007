AUTO-CRUISE CONTROL SWITCH CHECK


M11702000012ENG0000010000


1. Remove the auto-cruise control switch. (Refer to ).

2. Measure the resistance between terminal 3 and terminal 4 when each of the ON/OFF (MAIN), CANCEL, COAST/SET and ACC/RES switches is pressed. If the values measured at the time each switch is pressed correspond to those in the table below, the resistance values are correct.
