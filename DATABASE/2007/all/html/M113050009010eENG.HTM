FUEL FILTER PIPE ASSEMBLY REPLACEMENT <4M4>


M11305000901ENG0000010000


<SHORT WHEELBASE>


1. Remove the front scuff plate (R.H.) and outer seat belt (front) anchor bolt (R.H.), and then turn the floor carpet [Refer to GROUP 52A - Trim <Short Wheelbase> ].

2. Remove the service hole cover.

3. Disconnect the fuel main hose and fuel return hose.
4. Remove the mounting nuts and retainer plate.
5. While removing the fuel filter pipe assembly from the fuel tank, disconnect the suction hose from the fuel filter pipe assembly to remove the fuel filter pipe assembly from the service hole.
6. While inserting the fuel filter pipe assembly into the fuel tank, connect the suction hose to the fuel filter pipe assembly to install the fuel filter pipe assembly to the fuel tank.
7. Install the retainer plate to the fuel tank and tighten the mounting nuts.
8. Connect the fuel main hose and fuel return hose.
9. Install the service hole cover.
10. Spread the floor carpet, ant then install the front scuff plate (R.H.) and outer seat belt (front) anchor bolt (R.H.) [Refer to GROUP 52A - Trim <Short Wheelbase> ].

<LONG WHEELBASE>


1. Tumble the second seat (R.H.).

2. Remove the service hole cover (upper side).

3. Remove the service hole cover (lower side).

4. Disconnect the fuel main hose and fuel return hose.
5. Remove the mounting nuts and retainer plate.
6. While removing the fuel filter pipe assembly from the fuel tank, disconnect the suction hose from the fuel filter pipe assembly to remove the fuel filter pipe assembly from the service hole.
7. While inserting the fuel filter pipe assembly into the fuel tank, connect the suction hose to the fuel filter pipe assembly to install the fuel filter pipe assembly to the fuel tank.
8. Install the retainer plate to the fuel tank and tighten the mounting nuts.
9. Connect the fuel main hose and fuel return hose.
10. Install the service hole cover (lower side).
11. Install the service hole cover (upper side).
12. Restore the second seat (R.H.).
