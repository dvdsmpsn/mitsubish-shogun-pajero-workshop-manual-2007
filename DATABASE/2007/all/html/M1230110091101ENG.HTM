DIAGNOSIS CODE SET CONDITIONS


If the drive terminal voltage of the second solenoid valve is 3.0 V or less, it is judged that there is a short circuit or open circuit in the second solenoid valve, and diagnosis code No.P1775 (P0760) is output.
If diagnosis code No.P1775 (P0760) is output 4 times, the transmission is fixed in 3rd as a fail-safe measure.

PROBABLE CAUSES



  • Malfunction of second solenoid valve
  • Damaged harness wires and connectors
  • Malfunction of the engine-A/T-ECU
