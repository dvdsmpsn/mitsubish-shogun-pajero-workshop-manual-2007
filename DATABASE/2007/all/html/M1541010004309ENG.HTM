cautionIf the combination meter needs to be replaced, the current driving distance and elapsed days must be entered into the meter after the replacement in order to be used for service reminder function. Therefore, read "Mileage until reminder," "Months until reminder," "Mileage until Extra reminder," "Months until Extra reminder," and "Current schedule" from the meter before the replacement using the special function of M.U.T.-III, and note them. For the operation method of M.U.T.-III, refer to On-vehicle service . If "Mileage until reminder" or "Months until reminder" cannot be read from the meter using M.U.T.-III, use the following method.

  • As for the driving distance for check warning, use the driving distance displayed on the odometer.
  • As for the elapsed days for check warning, calculate the number of elapsed days from the delivery date to the customer (service reminder function start date) and current date.
