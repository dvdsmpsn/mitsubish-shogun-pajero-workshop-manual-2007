ACTIVE STABILITY CONTROL INDICATOR AND ACTIVE STABILITY CONTROL OFF INDICATOR INSPECTION


M13506200008ENG0000010000


Check whether Active Stability Control indicator and Active Stability Control OFF indicator illuminates in the following way.

1. When the ignition switch is turned to the "ON" position, the Active Stability Control indicator and Active Stability Control OFF indicator illuminates for approximately 3 seconds and then switches off.
2. When the ignition switch is turned to the START position, the Active Stability Control indicator and Active Stability Control OFF indicator go off.
3. When the ignition switch returns from the START to the ON position, the Active Stability Control indicator and Active Stability Control OFF indicator illuminates* for approximately 3 seconds, and then remains switched off.
4. If the illumination is other than the above, check the diagnosis codes.
