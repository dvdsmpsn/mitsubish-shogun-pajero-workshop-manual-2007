PRECAUTIONS ON HOW TO USE THE HEADLAMP ASSEMBLY


Be careful with the following items as resin lenses are used in the headlamp assembly.

  • Don't illuminate the headlamp for three minutes or more when the headlamp is covered with scratch protector.
  • Don't tape the outer lens.
  • Don't scratch the outer lens surface with a sharp edged special tool.
  • Use the specified wax remover and rinse the wax off cleanly.
  • Use the specified genuine bulb.
