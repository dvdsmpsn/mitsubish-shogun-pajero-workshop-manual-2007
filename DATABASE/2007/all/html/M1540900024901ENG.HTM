Code No.05 Communication Error with Power Window Main Switch


M15409000249ENG0000010000


cautionIf diagnosis code No.05 is set in the ETACS-ECU, diagnose the CAN bus lines.


TROUBLE JUDGMENT


The ETACS-ECU communicates with the power window main switch through the SWS communication lines. If there is any trouble in that communication, diagnosis code No.05 will be set.

COMMENTS ON TROUBLE SYMPTOM


If the power window main switch, the ETACS-ECU or the CAN bus line fails, this diagnosis code will be set.

PROBABLE CAUSES



  • Malfunction of the power window main switch
  • Malfunction of the ETACS-ECU switch
  • Malfunction of the CAN bus

DIAGNOSIS PROCEDURE

Use the M.U.T.-III to diagnose the CAN bus lines.
