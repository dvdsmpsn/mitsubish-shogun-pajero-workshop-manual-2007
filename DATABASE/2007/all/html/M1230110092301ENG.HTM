Code No.U1120: Failure information of engine-ECU


M12301100923ENG0000010000


DIAGNOSIS CODE SET CONDITIONS


When the received signal contains failure information, the diagnosis code No. U1120 is stored.

PROBABLE CAUSES



  • Damaged wiring harness or connector
  • Malfunction of the engine-A/T-ECU

DIAGNOSIS PROCEDURE


STEP 1. M.U.T.-III CAN bus diagnostics


Using M.U.T.-III, perform CAN busline diagnosis.

Q. Is the check result normal?

Go to Step 2.
Repair the CAN busline (Refer to GROUP 54D - Troubleshooting ). and then go to Step 5.

STEP 2. M.U.T.-III other system diagnosis codes


Check whether the MPI system diagnosis codes are set or not.

Q. Is the diagnosis code set?

Diagnose the MPI system (Refer to GROUP 13Ab - Troubleshooting, Inspection Chart For Diagnosis Code ).
Go to Step 3.

STEP 3. M.U.T.-III diagnosis code


Reconfirm whether the diagnosis codes are output from engine-A/T-ECU.
(1)Erase the diagnosis codes being output.
(2)Confirm whether the diagnosis codes are output.

Q. Is the diagnosis code output?

Replace engine-A/T-ECU. Then go to Step 4.
Intermittent malfunction in CAN busline (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points ).

STEP 4. M.U.T.-III diagnosis code


Reconfirm whether the diagnosis codes are output from engine-A/T-ECU.
(1)Erase the diagnosis codes being output.
(2)Confirm whether the diagnosis codes are output.

Q. Is the diagnosis code output?

Replace engine-A/T-ECU. Then go to Step 5.
The procedure is complete.

STEP 5. M.U.T.-III diagnosis code


Reconfirm whether the diagnosis codes are output from engine-A/T-ECU.

  • Erase the diagnosis codes being output.
  • Confirm whether the diagnosis codes are output.

Q. Is the diagnosis code output?

Go to Step 1.
The procedure is complete.
