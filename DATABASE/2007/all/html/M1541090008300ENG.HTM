Code No. 23 Height Sensor Power Supply Short to Earth


M15410900083ENG0000010000


cautionWhenever the ECU is replaced, ensure that the communication circuit is normal.


TROUBLE JUDGMENT


If the front height sensor, rear height sensor power supply, or earth line is short-circuited, diagnosis code No. 23 is set.

COMMENTS ON TROUBLE SYMPTOM


The front height sensor, the rear height sensor, the headlamp automatic levelling-ECU, or wiring harness and connectors between the front/rear height sensor and the headlamp automatic levelling-ECU may be defective.

PROBABLE CAUSES



  • Malfunction of the front height sensor
  • Malfunction of the rear height sensor
  • Malfunction of the headlamp automatic levelling-ECU
  • Damaged wiring harness and connectors

DIAGNOSIS PROCEDURE


STEP 1. Connector check: A-22 Front height sensor connector, E-117 headlamp automatic levelling-ECU connector


Q. Is the check result normal?

Go to Step 2.
Repair the defective connector.

STEP 2. Check the wiring harness from A-22 front height sensor connector (terminal No. 1 and 3) to D-125 headlamp automatic levelling-ECU connector (terminal No. 15 and 19).






  • Check the power supply lines and the earth lines for short circuit.


notePrior to the wiring harness inspection, check intermediate connector D-239, and repair if necessary.

Q. Is the check result normal?

Go to Step 3.
Repair the wiring harness.

STEP 3. Check whether the diagnosis code is reset.


Q. Is diagnosis code No. 23 set?

Go to Step 4.
A poor connection, open circuit, or other intermittent malfunctions between the front height sensor and the headlamp automatic levelling ECU. (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points - How to Cope with Intermittent Malfunctions .)

STEP 4. Connector check: E-117 Rear height sensor connector, D-125 headlamp automatic levelling-ECU connector





Q. Is the check result normal?

Go to Step 5.
Repair the defective connector.

STEP 5. Check the wiring harness from E-117 rear height sensor connector (terminal No. 1 and 3) to D-125 headlamp automatic levelling-ECU connector (terminal No. 16 and 20).






  • Check the power supply lines and the earth lines for short circuit.


notePrior to the wiring harness inspection, check intermediate connectors E-120, D-221 and repair if necessary.

Q. Is the check result normal?

Go to Step 6.
Repair the wiring harness.

STEP 6. Check whether the diagnosis code is reset.


Q. Is diagnosis code No. 23 set?

Go to Step 7.
A poor connection, open circuit, or other intermittent malfunctions between the rear height sensor and the headlamp automatic levelling-ECU. (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points - How to Cope with Intermittent Malfunctions .)

STEP 7. Check the front height sensor.


Check that the front height sensor works normally (Refer to ).

Q. Is the check result normal?

Go to Step 8.
Replace the front height sensor.

STEP 8. Check the rear height sensor.


Check that the rear height sensor works normally (Refer to ).

Q. Is the check result normal?

Go to Step 9.
Replace the rear height sensor.

STEP 9. Check whether the diagnosis code is reset.


Q. Is diagnosis code No. 23 set?

Replace the headlamp automatic levelling-ECU.
The diagnosis is complete.
