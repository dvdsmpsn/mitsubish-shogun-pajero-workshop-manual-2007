DISASSEMBLY SERVICE POINT


<<A>> BALL BEARING INSTALLATION



Using special tool Bearing remover (MD998801), support the ball bearing and remove it.

REASSEMBLY SERVICE POINTS


>>A<< BALL BEARING INSTALLATION



Using special tools to install the ball bearing.

  • Installer cup (MD998812)
  • Installer-200 (MD998814)
  • Installer adapter (54) (MD998826)

>>B<< SNAP RING INSTALLATION



Select a snap ring which will make the main drive pinion bearing end play equal to the standard value.
Standard value: 0 - 0.06 mm
