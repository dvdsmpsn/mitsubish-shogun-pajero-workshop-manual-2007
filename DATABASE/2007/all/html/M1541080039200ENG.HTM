Inspection Procedure 4: "-" is Displayed on the Driving Information Display or the Display is not Normal.


M15410800392ENG0000010000


COMMENTS ON TROUBLE SYMPTOM


The CAN bus line wiring harness, connector(s), or the engine-ECU <M/T> or engine-A/T-ECU <A/T>, combination meter may be defective. But it is no problem that "-" is displayed when the average fuel consumption display or the possible cruising distance display is reset.

PROBABLE CAUSES



  • Malfunction of multi centre display
  • Malfunction of combination meter
  • Damaged wiring harness and connectors

DIAGNOSIS PROCEDURE


STEP 1. M.U.T.-III CAN bus diagnostics

Use the M.U.T.-III to diagnose the CAN bus lines.

Q. Is the check result satisfactory?

Go to Step 2.
Repair the CAN bus lines. (Refer to GROUP 54D - Troubleshooting .) After diagnosing the CAN bus lines, go to Step 4.
STEP 2. M.U.T.-III other system diagnosis code

Check if an combination meter diagnosis code is set.

Q. Is the diagnosis code set?

Refer to GROUP 54Ab - Troubleshooting , and then go to Step 4.
Replace multi centre display, and then go to Step 3.
STEP 3. M.U.T.-III other system diagnosis code

Check if an MPI system diagnosis code is set.

Q. Is the diagnosis code set?

Refer to GROUP 13Ab - Troubleshooting <3800> or GROUP 13Bb - Troubleshooting , and then go to Step 4.
Replace multi centre display, and then go to Step 4.
STEP 4. Retest the system

Check if the driving information displays normally.

Q. Is the check result satisfactory?

The procedure is complete.
Go to Step 1.
