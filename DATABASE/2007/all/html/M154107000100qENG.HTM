Code No.B1703: Key ID inconsistent


M15410700010ENG0000010000


cautionWhenever the ECU is replaced, ensure that the input and output signal circuits are normal.

TROUBLE JUDGMENT


When the key ID that is sent from the transponder does not correspond to the key ID registered in the immobilizer-ECU, the immobilizer-ECU sets diagnosis code No.B1703.
noteDiagnosis code No.B1703 is always set along with diesel fuel system diagnosis code No.P0513.

PROBABLE CAUSES



  • Ignition key (key ID) is not registered
  • Malfunction of immobilizer-ECU

DIAGNOSIS PROCEDURE


Register all the ignition key key IDs again, and recheck the diagnosis code.

(1)Register all the ignition keys again (Refer to ).
(2)Erase the diagnosis code.
(3)The ignition switch is turned from the "LOCK" (OFF) position to the "ON" position with all ignition keys registered again.
(4)Check that diagnosis code No.B1703 is not reset.

Q. Is diagnosis code No.B1703 set?

Replace the immobilizer-ECU and register the ID codes (Refer to ). After registering the ID codes, check if the diagnosis code is set.
The diagnosis is complete. If another diagnosis code is set, refer to Diagnosis Code Chart .
