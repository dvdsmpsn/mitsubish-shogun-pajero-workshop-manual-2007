Inspection Procedure F-5: The Headlamp Automatic Levelling Warning Lamp does not Go Out.


M15410900111ENG0000010000


cautionWhenever the ECU is replaced, ensure that the power supply circuit, the earth circuit and the communication circuit are normal.

COMMENTS ON TROUBLE SYMPTOM


If the headlamp automatic levelling warning lamp does not go out, the diagnosis code may be set, or the automatic levelling-ECU or the combination meter may be defective.

PROBABLE CAUSES



  • Diagnosis code setting is under way.
  • Malfunction of the headlamp automatic levelling-ECU
  • Defective combination meter
  • Damaged wiring harness and connectors

DIAGNOSIS PROCEDURE


STEP 1. Check the diagnosis code.


Check if the diagnosis code is set.

Q. Is the diagnosis code set?

Perform troubleshooting at the diagnosis code setting.(Refer to .)
Go to Step 2.

STEP 2. M.U.T.-III actuator test


Perform the actuator test for the combination meter, and check that the headlamp automatic levelling warning lamp goes out (Refer to GROUP 54Ab - Troubleshooting ).

Q. Is the check result normal?

Go to Step 3.
Replace the combination meter.

STEP 3. Retest the system.


Check that the headlamp automatic levelling warning lamp goes out.

Q. Is the check result normal?

Intermittent malfunction. (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points - How to Cope with Intermittent Malfunctions .)
Replace the headlamp automatic levelling-ECU.
