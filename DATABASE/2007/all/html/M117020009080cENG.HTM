Code No. P1574 Engine-ECU and Its Related Components <4M4>


M11702000908ENG0000010000


DIAGNOSIS CODE SET CONDITIONS


This diagnosis code is set when there is an failure in the engine-ECU and its related components.

PROBABLE CAUSES



  • Malfunction of the diesel fuel system.
  • Malfunction of the engine-ECU.

DIAGNOSIS PROCEDURE


STEP 1. Check the diesel fuel system diagnosis code.


Refer to GROUP 13Bb - Diagnosis Function .

Q. Is any diagnosis code set?

Repair the diesel fuel system. (Refer to GROUP 13Bb - Inspection Chart for Diagnosis Code ). Then go to Step 3.
Go to Step 2.

STEP 2. Read the diagnosis code.


Q. Is diagnosis code No. P1574 set?

Replace the engine-ECU (Refer to GROUP 13Ba - Engine-ECU ). Then go to Step 3.
It can be assumed that this malfunction is intermittent (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunctions ).

STEP 3. Read the diagnosis code.


Q. Is diagnosis code No. P1574 set?

Return to Step 1 .
This procedure is complete.
