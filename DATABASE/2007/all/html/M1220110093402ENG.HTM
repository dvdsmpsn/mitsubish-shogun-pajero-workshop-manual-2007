Code No.61: ASTC-ECU system


M12201100934ENG0000010000





OPERATION


If the ASTC (active skid and traction control) system is activated, the ASTC-ECU sends a signal to the transfer-ECU. The transfer-ECU illuminates the relevant wheel of the 4WD indicator light according to the signal.

DIAGNOSIS CODE SET CONDITIONS


If there is a communication error with the ASTC-ECU (i.e. simultaneous open circuit in all signal lines), code No.61 is set.

PROBABLE CAUSES



  • Damaged harness wires and connectors
  • Malfunction of the ASTC system
  • Malfunction of the transfer-ECU

DIAGNOSIS PROCEDURE


STEP 1. Check the ASTC system.


Refer to GROUP 35C - Troubleshooting, Inspection Chart for Diagnosis Code .

Q. Is the check result normal?

Go to Step 2.
Repair or replace the ASTC system component(s).

STEP 2. Measure the voltage at transfer-ECU connector D-124.


(1)Turn the ignition switch to the ON position.


(2)Measure the voltage between transfer-ECU connector D-124 terminal No.6, 8, 10, 19 and earth.
OK: 1 V or less

Q. Is the check result normal?

Go to Step 5.
Go to Step 3.

STEP 3. Connector check: D-223 ASTC-ECU connector, D-225 intermediate connector.






Check for the contact with terminals.

Q. Is the check result normal?

Go to Step 4.
Repair the defective connector.

STEP 4. Check the harness between transfer-ECU connector D-124 (terminal No.6, No.8, No.10 and No.19) and ASTC-ECU connector D-223 (terminal No.73, No.71, No.72 and No.79).






Check the output line for open circuit.

Q. Is the check result normal?

Go to Step 5.
Repair the wiring harness.

STEP 5. Connector check: D-124 transfer-ECU connector.




Check for the contact with terminals.

Q. Is the check result normal?

Go to Step 6.
Repair the defective connector.

STEP 6. Replace the transfer-ECU.


(1)Replace the transfer-ECU.
(2)Test drive the vehicle.
(3)Check if the diagnosis code is set.

Q. Is diagnosis code 61 set?

Go to Step 1.
The procedure is complete.
