Inspection Procedure 2: When the Selector Lever is Moved to "N" Range, Auto-cruise Control is not Cancelled <6G7>.


M11702000902ENG0000010001


COMMENTS ON TROUBLE SYMPTOM


The inhibitor switch circuit is suspected.

PROBABLE CAUSES



  • Damaged harness or connector.
  • Malfunction of the inhibitor switch.
  • Malfunction of the engine-A/T-ECU.

DIAGNOSIS PROCEDURE


Refer to GROUP 23A - Troubleshooting <A/T>, Diagnosis Code Procedures Code No. P1770: Inhibitor Switch System (Open Circuit) , Code No. P1771: Inhibitor Switch System (Short Circuit) .
