A multi-link type double wishbone independent suspension has been adopted as the rear suspension. Compared with the conventional PAJERO, the following features have been improved.

  • The 17-inch brake has been adopted to improve the rigidity of the axle installation area.<Long wheelbase>
  • The wheel tread has been widened by 10 mm for cornering ability.<Long wheelbase>
  • The damping force of the shock absorber has been changed for handling stability and riding comfort.
  • The stabilizer has been changed for cornering ability.
  • The characteristic of the upper arm bushing has been changed for handling stability and riding comfort.
