Code No. 23 Engine-A/T-ECU and Its Related Components <6G7>


M11702000039ENG0000010002


DIAGNOSIS CODE SET CONDITIONS


This diagnosis code is set when there is an failure in the engine-A/T-ECU and its related components.

PROBABLE CAUSES



  • Malfunction of the MPI system.
  • Malfunction of the A/T system.
  • Malfunction of the engine-A/T-ECU.

DIAGNOSIS PROCEDURE


STEP 1. Check the MPI system diagnosis code.


Refer to GROUP 13Ab - Diagnosis Function .

Q. Is any diagnosis code set?

Repair the MPI system. (Refer to GROUP 13Ab - Inspection Chart for Diagnosis Code ). Then go to Step 4.
Go to Step 2.

STEP 2. Check the A/T system diagnosis code.


Refer to GROUP 23A - Troubleshooting <A/T>, Diagnosis Function .

Q. Is any diagnosis code set?

Repair the A/T system (Refer to GROUP 23A - Troubleshooting <A/T>, Diagnosis Code Chart ). Then go to Step 4.
Go to Step 3.

STEP 3. Read the diagnosis code.


Q. Is diagnosis code No. 23 set?

Replace the engine-A/T-ECU (Refer to GROUP 13Aa - Engine-A/T-ECU ). Then go to Step 4.
It can be assumed that this malfunction is intermittent (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunctions ).

STEP 4. Read the diagnosis code.


Q. Is diagnosis code No. 23 set?

Return to Step 1 .
This procedure is complete.
