The Mitsubishi multi-communication system (HDD navigation system with 7-inch wide monitor) incorporates a 30-GB hard disk. Moreover, the wide 2-DIN size 7-inch LCD display is adopted.
Screen (function)
Contents
Navigation screen
Displays navigation functions such as road map, search, guidance and information retrieval.
Screen of current vehicle location
Information on vehicle's current position (such as latitude, longitude, altitude and GPS reception status)
CD screen
Plays a CD (including CD-R/CD-RW) loaded in the CD drive.
MP3/WMA screen
Plays MP3/WMA data.
DVD screen
Plays a DVD loaded in the DVD drive.
Music server screen
Plays music recorded in the hard disk drive and records a music CD.
Rear monitor screen
Displays image via a DVD drive-incorporated rear seat 9-inch wide LCD monitor.
Radio screen
Displays available broadcast stations and channels.
Rear view camera screen
Displays rear view camera image.

Navigation function



  • The system can load a large-size map data.
  • It is compatible with nine languages (English, Germany, French, Spanish, Italian, Portuguese, Dutch, Swedish and Danish).
  • Most sophisticated LSI has been used as one for car navigation system.
    note*LSI stands for Large Scale Integration, and is a large-scale integrated circuit (IC) containing between 1000 and 100,000 circuit elements.
  • It incorporates a TMC-dedicated (real time traffic information function) tuner.
  • A voice guidance speaker can be selected in accordance with R.H. drive vehicles and L.H. drive vehicles.
