DISASSEMBLY SERVICE POINT


<<A>> SUPPLY PUMP GEAR REMOVAL


1. Fix the supply pump assembly using a vice.

2. Using special tool supply pump gear stopper (MB992096), fix the supply pump gear.
3. Remove the supply pump gear mounting nut.

4. Using special tool side bearing puller (MB990810), remove the supply pump gear.

ASSEMBLY SERVICE POINT


>>A<< SUPPLY PUMP GEAR INSTALLATION


1. Fix the supply pump assembly using a vice.
2. Install the supply pump gear to the supply pump assembly.

3. Using special tool supply pump gear stopper (MB992096), fix the supply pump gear.
4. Install the supply pump gear mounting nut, and then tighten to the specified torque.
Tightening torque: 64 ± 5 N·m
