FUEL TANK GAUGE UNIT CHECK


M15410100040ENG0000010000


Remove the fuel tank gauge unit (Refer to GROUP 13C, Fuel Tank <3,800> or <3,200>).

FUEL TANK GAUGE UNIT RESISTANCE




1. Check that the resistance value between the fuel tank gauge unit terminal and earth terminal is at the standard value when the float of the fuel tank gauge unit is at the F position and the E position.
Standard values:
Float Position
Gauge resistance value
F position
3.0 ± 0.6 Ω
E position
110.0 ± 2.6 Ω
2. Check that the resistance value changes smoothly when the float moved slowly between the F position and the E position.
3. If all checks are correct, go to fuel tank gauge unit float height check. If any check is not correct, replace the fuel tank gauge unit.

FUEL TANK GAUGE UNIT FLOAT HEIGHT




1. Move the float and check that F position height (A) and E position height (B) are at the standard values when the float arm touches the stopper.
Standard values <Short wheelbase>:
Float Position
Float Centre Height
F position (A)
11.9 ± 3.0 mm
E position (B)
195.2 ± 3.0 mm
Standard values <Long wheelbase>:
Float Position
Float Centre Height
F position (A)
11.5 ± 3.0 mm
E position (B)
204.9 ± 3.0 mm
2. If all checks are correct, go to fuel tank gauge unit float height check. If any check is not correct, replace the fuel tank gauge unit.
