Inspection Procedure 12: The Feeling of Impact or Vibration when Decelerating


M11310100065ENG0000010000


COMMENTS ON TROUBLE SYMPTOM



  • Failure is possibly caused by an insufficient amount of intake air due to failed electronic-controlled throttle valve system.

PROBABLE CAUSES



  • Failed electronic control throttle valve system
  • Contamination around the throttle valve
  • Failed engine-A/T-ECU

DIAGNOSIS PROCEDURE


STEP 1. M.U.T.-III diagnosis code

Inspection chart for diagnosis code (Refer to ).

    4. Item 15: Throttle position sensor (sub)
