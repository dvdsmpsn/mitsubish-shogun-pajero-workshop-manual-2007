INJECTOR


The injector is a solenoid control type that injects fuel in accordance with the injection signals sent by the engine-ECU. An injector, which is provided for each cylinder, is mounted in the cylinder head.
The injection characteristics of injectors vary individually. Therefore, a 30-digit alphanumeric ID code is stamped on each injector. Engine-ECU provides a fuel injection amount that suits the individual injection characteristics, based on these ID codes.

STRUCTURE AND SYSTEM



An injector consists of a nozzle, command piston, command chamber, and solenoid valve that controls the ON/OFF of the outflow orifice. The command piston moves vertically in accordance with the fuel pressure in the command chamber. As the nozzle is integrated with the command piston, the orifice of the injector opens and closes with the vertical movement of the command piston.




When the solenoid valve is OFF, the two-way valve pushes down, causing the outflow orifice of the command chamber to close. Because the command chamber fills with pressurized fuel, the nozzle pushes down together with the command piston. As a result, no injection of fuel takes place.
When the solenoid valve is ON, the two-way valve pulls up, causing the outflow orifice of the command chamber to open. The pressure in the command chamber decreases, causing the nozzle to pull up together with the command piston. This results in the injection of fuel.
When the solenoid valve is turned OFF, the two-way valve closes, immediately pushing down the nozzle and closing the fuel injection orifice. An injector can perform these ON/OFF operations within an extremely short length of time. Therefore, as the solenoid valve is turned ON/OFF two or more times, the injector can inject fuel in multiple stages according to the number of times it is turned ON/OFF.


THROTTLE VALVE CONTROL SERVO



A throttle valve control servo is installed in throttle body.
The throttle valve control servo performs the Open/Close of the throttle valve through the reduction gear.
Engine-ECU changes current direction according to the Open/Close direction and also changes current to the motor coil in order to the control throttle valve control servo.
Throttle valve control servo is composed of a good response, low energy, and small DC motor with brush and can generate rotation force corresponding to the current applied on the coil.



SWIRL CONTROL SOLENOID VALVE



A swirl control solenoid valve is installed on the rocker cover.
The swirl control solenoid valve is an ON/OFF control type solenoid valve.
When current is not passing through the coil, nipple A is kept air-tight and air passes through nipple B and the filter.
When current is passed through the coil, air is unable to pass through the filter and it passes through nipple A and nipple B.
Engine-ECU changes the ON/OFF of solenoid valve according to the engine's operating conditions to change between vacuum pressure and atmospheric pressure and performs the Open/Close of the swirl control valve.
