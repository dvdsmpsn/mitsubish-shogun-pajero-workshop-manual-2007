POWER WINDOW TIMER FUNCTION CHECK


M14203000904ENG0000010000


Check the sunroof timer as described below. If it does not work, perform troubleshooting. Refer to .

  • Close the door and turn the ignition switch to the LOCK (OFF) position, and then check that the power windows operate for 30 seconds.
  • The timer activates by closing the doors and turning the ignition switch to the LOCK (OFF) position. Check if the power windows operates within 30 seconds after the driver's door is opened while the timer is being activated. The timer period will be extended if the driver's door is opened, but the timer will be cancelled if the driver's door is closed while the timer is being activated.
