IDLE SPEED CHECK


M11102000912ENG0000010001


1. Set the vehicle to the pre-inspection condition.

2. Turn the ignition switch to "LOOK" (OFF) position, and connect the diagnosis connector to the M.U.T.-III.
3. Start the engine, and let it run at idle.
4. Check the idle speed.
Standard value: 700 ± 30 r/min
5. If the idle speed is not within the standard value, inspect the diesel system (Refer to GROUP 13B - Troubleshooting - Inspection Chart for Diagnosis Code ).
