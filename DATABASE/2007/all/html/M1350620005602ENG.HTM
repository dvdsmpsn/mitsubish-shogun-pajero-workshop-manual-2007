Code No.81: G-sensor Calibration Incomplete
Code No.82: Yaw Rate Sensor Calibration Incomplete


M13506200056ENG0000010000


cautionIf there is any problem in the CAN bus lines, an incorrect diagnosis code may be set. Diagnose the CAN bus lines before the diagnosis codes (Refer to GROUP 54D, CAN bus-line diagnostic flow ).

DIAGNOSIS CODE SET CONDITIONS


When the G and yaw rate sensor or ASTC-ECU is replaced with new one, G and yaw rate sensor calibration must be performed. Diagnosis codes No. 81 and 82 are set when the calibration is not performed after the ASTC-ECU is replaced.
noteBy performing calibration, the calibrated value in the ASTC-ECU is updated. Calibration must be performed when a new ASTC-ECU is used. The calibration state is stored in the ASTC-ECU. When only the G and yaw rate sensor is replaced, the diagnosis code for "calibration incomplete" is not set, but the calibration must be performed. Also when the ASTC-ECU that has already been calibrated on another vehicle is removed and used, calibration must be performed.

PROBABLE CAUSES


The most likely causes for these diagnosis codes to set are:

  • Damaged wiring harness or connector
  • G and yaw rate sensor calibration not performed
  • Malfunction of the G and yaw rate sensor
  • Malfunction of the ASTC-ECU

DIAGNOSIS


STEP 1. M.U.T.-III CAN bus diagnostics


cautionBefore connecting or disconnecting the M.U.T.-III, turn the ignition switch to the "LOCK" (OFF) position.
(1)Connect M.U.T.-III to the 16-pin diagnosis connector.
(2)Turn the ignition switch to the "ON" position.
(3)Diagnose the CAN bus line.
(4)Turn the ignition switch to the "LOCK" (OFF) position.

Q. Is the check result normal?

Go to Step 3.
Repair the CAN bus line (Refer to GROUP 54D, CAN bus line Diagnostic flow ). Then go to Step 2.

STEP 2. Check whether the diagnosis code is reset.


cautionBefore connecting or disconnecting the M.U.T.-III, turn the ignition switch to the "LOCK" (OFF) position.
(1)Turn the ignition switch to the "ON" position.
(2)Erase the diagnosis code.
(3)Turn the ignition switch to the "LOCK" (OFF) position.
(4)Turn the ignition switch to the "ON" position.
(5)Check if the diagnosis code is set.
(6)Turn the ignition switch to the "LOCK" (OFF) position.

Q. Is code No.81 and 82 set?

Go to Step 3.
The procedure is complete.

STEP 3. Check the diagnosis code again after the G and yaw rate sensor calibration.


cautionBefore connecting or disconnecting the M.U.T.-III, turn the ignition switch to the "LOCK" (OFF) position.
(1)Turn the ignition switch to the "ON" position.
(2)Perform the G and yaw rate sensor calibration.(Refer to .)
(3)Turn the ignition switch to the "LOCK" (OFF) position.
(4)Turn the ignition switch to the "ON" position.
(5)Check if the diagnosis code is set.
(6)Turn the ignition switch to the "LOCK" (OFF) position.

Q. Is code No.81 and 82 set?

Replace the G and yaw rate sensor. Then go to Step 4.
The procedure is complete.

STEP 4. Check the diagnosis code again after the G and yaw rate sensor calibration.


cautionBefore connecting or disconnecting the M.U.T.-III, turn the ignition switch to the "LOCK" (OFF) position.
(1)Turn the ignition switch to the "ON" position.
(2)Perform the G and yaw rate sensor calibration.(Refer to .)
(3)Turn the ignition switch to the "LOCK" (OFF) position.
(4)Turn the ignition switch to the "ON" position.
(5)Check if the diagnosis code is set.
(6)Turn the ignition switch to the "LOCK" (OFF) position.

Q. Is code No.81 and 82 set?

Replace the ASTC-ECU.
The procedure is complete.
