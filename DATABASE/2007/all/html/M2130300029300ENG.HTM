IDLE SPEED CONTROL

System Configuration Diagram

Engine-ECU compares the target idle speed with the actual idle speed, and controls the fuel injection amount so that the actual idle speed will attain the target idle speed (feedback control). Furthermore, the following compensations are added to control the engine at a stable idle speed.


List of main compensations for idle speed control
Compensations
Content
Engine coolant temperature compensation
This compensation is made to the target idle speed to suit the engine coolant temperature. The lower the engine coolant temperature, the higher the target idle speed will be.
Compensation when A/C is functioning
This compensation is made to the target idle speed to suit the operation of the A/C compressor. It sets the target idle speed higher when the A/C compressor is operating.
Compensation when PTC heater is functioning
This compensation is made to the target idle speed to suit the operation of the PTC heater. It sets the target idle speed higher when the PTC heater is operating.
