Code No.11: Transponder communication system or radio interference of key ID.


M15410700029ENG0000010000




DIAGNOSIS CODE SET CONDITIONS


This code is set if the key ID is sent to the immobilizer-ECU from the ignition key when the ignition switch is turned to the ON position.

PROBABLE CAUSES



  • Damaged harness wires and connectors
  • Malfunction of ignition key
  • Malfunction of immobilizer-ECU

DIAGNOSIS PROCEDURE


STEP 1. Check that the engine is started with another registered ignition key.

Q. Is the check result normal?

Replace the ignition key that cannot start the engine with new registered ignition key. (Refer to ).
Go to Step 2.
STEP 2. M.U.T.-III diagnosis code

Check that the immobilizer-ECU sets a diagnosis code.

Q. Is the diagnosis code set?

YES (diagnosis cede No.11 is set) : Replace the immobilizer-ECU and register the ignition key. (Refer to ).
YES (diagnosis cede No.12 is set) : Code No.12 key IDs are not the same or are not registered. (Refer to ).
NO (diagnosis code Nos.11 and 12 is not set) : The procedure is complete.
