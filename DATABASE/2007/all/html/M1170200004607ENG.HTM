Inspection Procedure 3: When the Auto-cruise Control CANCEL Switch is Set to ON, Auto-cruise Control is not Cancelled <6G7>.


M11702000046ENG0000010002


COMMENTS ON TROUBLE SYMPTOM


The cause is probably an open-circuit in the circuit inside the auto-cruise control CANCEL switch.

PROBABLE CAUSE


Malfunction of the auto-cruise control switch.

DIAGNOSIS PROCEDURE


Replace the auto-cruise control switch (Refer to ). Then check the malfunction is eliminated.
