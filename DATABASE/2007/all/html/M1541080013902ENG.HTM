Inspection Procedure 9: The Rear Display does not Change to Nighttime Brightness when the Headlamp is Illuminated.


M15410800139ENG0000010000




DIAGNOSIS PROCEDURE


STEP 1. M.U.T.-III other system diagnosis code


Check if the diagnosis code is set to the ETACS-ECU.

Q. Is the check result normal?

Troubleshoot the ETACS-ECU (Refer to ).
Go to Step 2.

STEP 2. Connector check: D-420 ETACS-ECU connector, E-129 rear display unit connector





Q. Is the check result normal?

Go to Step 3.
Repair the connector concerned.

STEP 3. Check the wiring harness wires between D-420 ETACS-ECU connector terminal No.60 and E-129 rear display unit connector terminal No.5






  • Check the communication line.


notePrior to the wiring harness inspection, check intermediate connector D-131, and repair if necessary.

Q. Is the check result normal?

Go to Step 4.
Repair the wiring harness.

STEP 4. Substitute a known good rear display unit, and check the trouble symptom.


Check whether rear monitor does change to nighttime brightness when the headlamps are illuminated.

Q. Is the check result normal?

Replace the rear display unit.
Replace the ETACS-ECU.
