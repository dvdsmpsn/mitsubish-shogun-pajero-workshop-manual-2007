EXHAUST PIPE AND MAIN MUFFLER

<6G7>


5-piece type of short wheelbase exhaust pipe which consists of the front exhaust pipe, front catalytic converter, rear catalytic converter, centre exhaust pipe and exhaust main muffler has been adopted.
5-piece type of long wheelbase exhaust pipe which consists of the front exhaust pipe, front catalytic converter, rear catalytic converter, exhaust main muffler and exhaust tail pipe has been adopted.

<4M4>


5-piece type of short wheelbase exhaust pipe which consists of the front catalytic converter, front exhaust pipe, rear catalytic converter, centre exhaust pipe and exhaust main muffler has been adopted.
5-piece type of long wheelbase exhaust pipe which consists of the front catalytic converter, front exhaust pipe, rear catalytic converter, exhaust main muffler and exhaust tail pipe has been adopted.
The exhaust system has the following features in the same way as the conventional PAJERO.

  • For the centre exhaust pipe, an acoustic pre-muffler is installed to reduce high frequency noise <Short wheelbase>.
  • The large muffler is installed to secure high-output of an engine and reduce exhaust noise.
  • For the exhaust tail pipe, an resonance pre-muffler is installed to reduce low frequency booming noise <Long wheelbase>.
  • The connection between the front catalytic converter and the rear catalytic converter has a low friction type spring joint structure, to reduce vibration of the exhaust pipe <6G7>.
  • The connection between the front exhaust pipe and the rear catalytic converter has a low friction type spring joint structure, to reduce vibration of the exhaust pipe <4M4>.
