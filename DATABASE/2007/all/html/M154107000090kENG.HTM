Code No.B1702: Reception error of key ID


M15410700009ENG0000010000


cautionWhenever the ECU is replaced, ensure that the input and output signal circuits are normal.

OPERATION


The key reminder switch (ignition key ring antenna) supplies the ignition key with power, and the ignition key sends the key ID to the immobilizer-ECU. Immobilizer-ECU receives the key ID with the key ring antenna, and judges if the ignition key is the registered one.

TROUBLE JUDGMENT



  • Diagnosis code No.B1702 is stored when there are other ignition keys or anything that interferes with the communication (things that generate radio waves such as magnets and an air-cleaning device that has a power plug) near the ignition key that is going to start the engine.
  • If the key ID is not sent to the immobilizer-ECU even when the ignition switch is turned to the ON position, diagnosis code No.B1702 is stored.
noteDiagnosis code No.B1702 is always set along with diesel fuel system diagnosis code No.P0513.

PROBABLE CAUSES



  • Interference of the key ID
  • Malfunction of ignition key
  • Malfunction of immobilizer-ECU

DIAGNOSIS PROCEDURE


STEP 1. Check the ignition key inserted in the key cylinder for interference.

Q. Are there other ignition keys or anything that interferes with the communication (things that generate radio waves such as magnets and an air-cleaning device that has a power plug) near the ignition key inserted in the key cylinder?

Move away or remove other ignition keys or things that interfere with the communication (things that generate radio waves such as magnets and a air-cleaning device that has a power plug) near the ignition key. Then, go to step 3.
Go to Step 2.
STEP 2. Check with the spare key that has already been registered.

Check that the engine starts using the spare key that has already been registered.

Q. Is diagnosis code No.B1702 set?

Replace the immobilizer-ECU and register the ID codes again (Refer to ). After registering the ID codes, go to Step 3.
Replace the ignition key for which diagnosis code No.B1702 is set, and register the ID codes again (Refer to ). After registering the ID codes, go to Step 3.
STEP 3. Check whether the diagnosis code is reset.

(1)Erase the diagnosis code.
(2)Turn the ignition switch to the "LOCK" (OFF) position once, and then turn it to the "ON" position again.
(3)On completion, check that diagnosis code No.B1702 is not reset.

Q. Is diagnosis code No.B1702 set?

Return to Step 1.
The diagnosis is complete. If another diagnosis code is set, refer to Diagnosis Code Chart .
