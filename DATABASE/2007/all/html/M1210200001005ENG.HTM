REASSEMBLY SERVICE POINTS


A CLUTCH RELEASE BEARING INSTALLATION



Apply the specified grease to the clutch release bearing inside, and fill grease in the groove.
Specified lubricants:
Mitsubishi Part No. 0101011 or equivalent

>>B<< RELEASE FORK AND RELEASE FORK SHAFT INSTALLATION



Apply the specified grease to contact surface of release fork and release fork shaft.
Specified lubricants:
Mitsubishi Part No. 0101011 or equivalent

>>C<< SEALING CAP INSTALLATION



Press fit the sealing cap inti the transmission as shown. It must be 2 mm below the housing surface.

>>D<< CLUTCH RELEASE CYLINDER INSTALLATION



Apply the specified grease to contact surface of clutch release cylinder and push rod.
Specified lubricants:
Mitsubishi Part No. 0101011 or equivalent

>>E<< CLUTCH DISC INSTALLATION



Apply the specified grease to the clutch disc splines and rub it in splines with a brush.
Specified lubricants:
Mitsubishi Part No. 0101011 or equivalent

>>F<< CLUTCH COVER INSTALLATION



Install the clutch cover, aligning its installation position with the dowel pins, and then tighten the bolts to the specified torque of 19 ± 3 N·m in the order shown in the illustration.
