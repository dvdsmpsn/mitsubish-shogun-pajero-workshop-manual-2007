Code No.P1790 Back up line system


M12301100917ENG0000010000




OPERATION


Always power is supplied to the ECU to prevent the reset of engine-A/T-ECU memory when the ignition switch is turned to the LOCK (OFF) position.

DIAGNOSIS CODE SET CONDITION


If the battery voltage is 10 V or more and back-up voltage is less than 6 V while the ignition switch is ON position for 2 seconds or more, it is judged that there is a open circuit in the back-up line and diagnosis code No. P1790 is set.

PROBABLE CAUSES



  • Damaged harness wires and connectors
  • Malfunction of the engine-A/T-ECU

DIAGNOSIS PROCEDURE


STEP 1. Measure the voltage at engine-A/T-ECU connector D-214.




(1)Measure the voltage between engine-A/T-ECU connector D-214 terminal No.58 and earth.

STEP 3. Check the harness between engine-A/T-ECU connector D-214 terminal No.58 and fusible link.

STEP 4. Connector check: D-214 engine-A/T-ECU connector.




Check for the contact with terminals.

STEP 5. Replace the engine-A/T-ECU and then recheck the diagnosis code.


(1)Replace the engine-A/T-ECU.
