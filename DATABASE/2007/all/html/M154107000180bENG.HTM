Inspection Procedure 3: The ignition key cannot be registered with the M.U.T.-III. <3,800>


M15410700018ENG0000010001


CIRCUIT OPERATION


Refer to .

COMMENTS ON TROUBLE SYMPTOM


If only some ignition keys cannot be registered, the ignition key itself may be defective. If no ignition key can be registered, the immobilizer-ECU may be defective.

PROBABLE CAUSES



  • Malfunction of ignition key
  • Damaged harness wires and connectors
  • Malfunction of immobilizer-ECU
  • Malfunction of engine-A/T-ECU

DIAGNOSIS PROCEDURE


STEP 1. Check that no ignition key can be registered.


Q. Can any of the ignition keys be registered?

Replace the ignition key that cannot be registered with new registered ignition key.
Go to Step 2.

STEP 2. Connector check: D-310 key reminder switch connector and D-216 engine-A/T-ECU connector






Q. Is the check result normal?

Go to Step 3.
Repair the defective connector.

STEP 3. Check the harness wires between D-310 key reminder switch connector terminal No.7 and D-216 engine-A/T-ECU connector terminal No.65.








notePrior to the wiring harness inspection, check intermediate connector D-229, and repair if necessary.

Q. Is the check result normal?

Go to Step 4.
Repair the wiring harness.

STEP 4. Check the communication with M.U.T.-III


Q. Can the system communicate with the M.U.T.-III?

Go to Step 5.
Refer to Inspection Procedure 1 "MPI system-related diagnostic code No.P0513 is set, and communication between M.U.T.-III and immobilizer cannot be established ".

STEP 5. M.U.T.-III diagnosis code


Check whether the engine immobilizer system-related diagnosis code is set.

Q. Is the diagnosis code set?

Refer to .
Go to Step 6.

STEP 6. Retest the system.


Check that the ignition key can be registered.

Q. Is the check result normal?

The procedure is complete.
Replace the immobilizer-ECU and register the ignition key (Refer to ). Then go to Step 7.

STEP 7. Retest the system.


Check that the ignition key can be registered.

Q. Is the check result normal?

The procedure is complete.
Replace the engine-A/T-ECU and register the ignition key (Refer to ).
