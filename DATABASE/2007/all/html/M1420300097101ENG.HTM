INSPECTION PROCEDURE 4: Encrypted Code cannot be Registered. <Using SWS monitor>


M14203000971ENG0000010001


cautionWhenever the ECU is replaced, ensure that the input and output signal circuits are normal.


COMMENTS ON TROUBLE SYMPTOM


If the encrypted code registration mode is not entered, the key reminder switch, the hazard warning lamp switch or the ETACS-ECU may be defective.
If the registration is not possible although the registration mode is entered, the keyless entry transmitter or the ETACS-ECU may be defective.

POSSIBLE CAUSES



  • Malfunction of the keyless entry transmitter
  • Malfunction of the ETACS-ECU
  • Damaged harness wires and connectors

DIAGNOSIS PROCEDURE


Step 1. Check the encrypted code registration mode.


Check that the encrypted code registration mode is entered.
