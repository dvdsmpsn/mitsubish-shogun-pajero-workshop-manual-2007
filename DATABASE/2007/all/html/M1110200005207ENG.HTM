REMOVAL SERVICE POINT


<<A>> VACUUM HOSE REMOVAL



Aligning with the marking on the vacuum hose, make mating marks on the vacuum pump assembly or the vacuum pipe and the hose clamp, and then remove the vacuum hose.

INSTALLATION SERVICE POINT


>>A<< VACUUM HOSE INSTALLATION



1. When the vacuum pump assembly is replaced with a new one, make a mating mark in the same position as the mating mark on the removed vacuum pump assembly.
2. As shown in the figure, install the vacuum hose and the hose clamp so that the marking is aligned with the mating mark.
