Inspection Procedure 1: When the Brake Pedal is Depressed, Auto-cruise Control is not Cancelled <6G7>.


M11702000041ENG0000010002


COMMENTS ON TROUBLE SYMPTOM


The stop lamp switch circuit is suspected.

PROBABLE CAUSES



  • Damaged harness or connector.
  • Malfunction of the stop lamp switch.
  • Malfunction of the engine-A/T-ECU.

DIAGNOSIS PROCEDURE


Refer to - Diagnosis Code Procedures <6G7>, Code No. 22: Stop Lamp Switch System <6G7>.
