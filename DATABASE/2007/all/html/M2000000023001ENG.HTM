FRONT SUSPENSION


Newly developed Independent-wishbone, coil springs has been adopted.



  • The coil spring has been adopted to reduce friction and optimise longitudinal stiffness for the purpose of improvement in riding comfort.
  • The ball-joint type stabilizer link has been adopted to reduce friction for riding comfort improvement and vehicle deflection reduction.
  • To improve maintenance performance, the resin-type ball joint bearing has been adopted for upper and lower arms, making grease replenishment unnecessary.
  • Due to the adoption of the rack and pinion type steering gear box, the suspension geometry has been optimised to improve straightforward driving stability and braking stability.
  • Lower arm cam bolt has been adopted to improve maintenance performance.
  • The suspension spec has been refined to improve road contact, driveability on poor road surfaces, critical turning performance and riding comfort.

REAR SUSPENSION


A multi-link type double wishbone independent suspension has been adopted as the rear suspension.



  • Wide tread has been adopted to improve steering ability.
  • The suspension geometry has been optimised to improve steering stability, turning performance and riding comfort.
  • Zero initial camber provides improved turning performance.
  • The suspension spec has been refined to improve road contact, driveability on poor road surfaces, critical turning performance and riding comfort.
