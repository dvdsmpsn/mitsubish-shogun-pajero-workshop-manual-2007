Code No.B1200 Malfunction odometer


M15410100009ENG0000010000


TROUBLE JUDGMENT


This code is stored when the odometer value cannot be determined when reading, writing or verifying the EEPROM storing the odometer value, or the odometer displays abnormal value.

COMMENTS ON TROUBLE SYMPTOM


The combination meter may be defective.

PROBABLE CAUSES


Malfunction of the combination meter

DIAGNOSIS PROCEDURE


Check whether the diagnosis code is reset.

Recheck if the diagnosis code is set.
(1)Erase the diagnosis code.
(2)Turn the ignition switch to "ON" position.
(3)Check if the diagnosis code is set.

Q. Is the check result satisfactory?

Replace combination meter.
The trouble can be an intermittent malfunction (Refer to GROUP 00 - How to use troubleshooting/Inspection Service Points ).
