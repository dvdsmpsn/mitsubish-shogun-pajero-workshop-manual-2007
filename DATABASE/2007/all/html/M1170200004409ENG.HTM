Inspection Procedure 6: When the Auto-cruise Control ON/OFF (MAIN) Switch is Turned ON, the Auto-cruise Control Indicator Lamp does not Illuminate. (However, the auto-cruise control system is normal) <4M4>.


M11702000044ENG0000010002




OPERATION


The engine-ECU detects ON/OFF (MAIN) switch ON signal to illuminate the auto-cruise control indicator lamp on the combination meter.

COMMENTS ON TROUBLE SYMPTOM


Connector(s), wiring harness in the CAN bus line between the engine-ECU and the combination meter, power supply to the engine-ECU, the combination meter, the engine-ECU may be defective.

PROBABLE CAUSES



  • Damaged harness or connector.
  • Malfunction of the combination meter
  • Malfunction of the engine-ECU.

DIAGNOSIS PROCEDURE


STEP 1. M.U.T.-III CAN bus diagnosis.



  • Using M.U.T.-III, perform CAN bus line diagnosis.

Q. Is the check result satisfactory?

Go to Step 2.
Repair the CAN bus lines (Refer to GROUP 54D - Troubleshooting, Can Bus diagnosis Table ). Then go to Step 4.

STEP 2. Check the combination meter system diagnosis code.


Refer to GROUP 54Ab - Troubleshooting, Diagnosis Function .

Q. Is any diagnosis code set?

Refer to GROUP 54Ab - Troubleshooting, Diagnosis Code Chart . Then go to Step 4.
Go to Step 3.

STEP 3. Retest the system.


Q. Does the auto-cruise control indicator lamp illuminate when the ON/OFF (MAIN) switch is turned ON?

It can be assumed that this malfunction is intermittent (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunctions ).
Replace the engine-ECU (Refer to GROUP 13Ba - Engine-ECU ). Then go to Step 4.

STEP 4. Retest the system.


Q. Does the auto-cruise control indicator lamp illuminate when the ON/OFF (MAIN) switch is turned ON?

The procedure is complete.
Return to Step 1.
