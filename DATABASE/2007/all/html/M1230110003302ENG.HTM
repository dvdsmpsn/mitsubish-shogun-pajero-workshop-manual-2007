SELECTOR LEVER POSITION INDICATOR LAMP SYSTEM



If there is a problem with any of the A/T system, the selector lever position indicator lamp will flash at a rate of approximately once per second.
If the selector lever position indicator lamp is flashing at a rate of approximately once per second, check the diagnosis output.
Selector lever position indicator lamp flashing item
  • Input shaft speed sensor system
  • Output shaft speed sensor system
  • Solenoid valves system
  • Non-synchronization at various shift ranges
  • A/T control relay system
noteIf the A/T fluid temperature warning lamp is illuminating, the A/T fluid temperature is high. (It flushes when the fluid is approximately 135°C or more and goes off when the fluid is approximately 115°C or less.)

METHOD OF READING THE DIAGNOSIS CODE


Use the M.U.T.-III to read the diagnosis code. (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, Diagnosis Function ).

METHOD OF ERASING THE DIAGNOSIS CODE


Use the M.U.T.-III to erase the diagnosis code. (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, Diagnosis Function ).

HOW TO INITIALISE A/T LEARNING VALUE


AIM


A/T learned value must be reset whenever the automatic transmission, engine assembly, A/T valve body, or A/T solenoid valve is replaced. It cannot be reset by disconnecting the battery. Use the M.U.T.-III as follows:

INITIALISATION PROCEDURE


1. Shift the selector lever to P and turn the ignition switch to the LOCK (OFF) position.
2. Connect the M.U.T.-III to the vehicle's diagnosis connector.
3. In the ELC-A/T menu screen, select "Special Function," then select "Memory Reset."
4. Select "OK" to reset the A/T learned memory.
5. After this initialisation, make the system learn the idling in accordance with "Learning procedure for idling in MPI engine" (Refer to GROUP 00 - Precautions before Service, Learning Procedure For Idling in MPI Engine ).
noteThis reset procedure will also automatically initialise the INVECS-II Learned Value. A/T diagnosis codes and A/T freeze-frame data will be erased. (Engine diagnosis codes, engine-related freeze-frame data, and Readiness status will remain even after A/T Learned Value is reset.)
