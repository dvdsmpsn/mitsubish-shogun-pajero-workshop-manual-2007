DISASSEMBLY SERVICE POINTS


<<A>> EDJ BOOT REMOVAL



Wrap plastic tape around the spline part on the EDJ side of the driveshaft so that EDJ boot is not damaged when removed.

REASSEMBLY SERVICE POINTS


>>A<< EDJ BOOT INSTALLATION


Install EDJ boot after wrapping plastic tape around the spline part on the driveshaft.

>>B<< EDJ BOOT/EDJ BOOT BAND INSTALLATION



1. Position the EDJ outer race so that the distance between the boot bands is at the standard value.
Standard value (A): 93 ± 5 mm (LH), 85 ± 5 mm (RH)
2. Remove part of the EDJ boot from the EDJ outer race to release the air pressure inside the boot.
