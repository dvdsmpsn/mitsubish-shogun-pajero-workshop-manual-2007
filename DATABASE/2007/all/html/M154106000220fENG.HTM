Inspection Procedure B-1: Lamp reminder buzzer function does not work normally.


M15410600022ENG0000010000


cautionWhenever the ECU is replaced, ensure that the input and output signal circuits are normal.


COMMENTS ON TROUBLE SYMPTOM


The ETACS-ECU operates this function in accordance with the input signals below.

  • Ignition switch (IG1)
  • Driver's door switch
  • Tail lamp switch
  • Headlamp switch
If this function does not work normally, these input signal circuit(s) or the ETACS-ECU may be defective.

PROBABLE CAUSES



  • Malfunction of the driver's door switch
  • Malfunction of the column switch
  • Malfunction of the ETACS-ECU
  • Damaged harness wires and connectors

DIAGNOSIS PROCEDURE


STEP 1. Check the power supply circuit.


When the ignition switch is turned to the LOCK (OFF) position, check if the hazard warning lamps illuminate.

Q. Is the check result normal?

Go to Step 2.
Refer to inspection procedure A-1 "Check the battery power supply circuit to the ETACS-ECU ."

Check the input signals below, which are related to the lamp reminder buzzer function.
System switch

Ignition switch (IG1)
When turned from ACC to ON
Driver's door switch
When the driver's door is opened
Tail lamp switch
When the lighting switch is turned to the TAIL position
Headlamp switch
When the lighting switch is turned to the HEADLAMP position
