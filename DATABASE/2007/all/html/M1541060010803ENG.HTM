Inspection Procedure C-16: The signal from the central door lock switch (incorporated with the power windows main switch) is not received.


M15410600108ENG0000010000


cautionWhenever the ECU is replaced, ensure that the input signal circuit is normal.


COMMENTS ON TROUBLE SYMPTOM


Input signal from the central door lock switch (incorporated with the power window main switch) and door lock key slangier are used to operate the central door locking function. If the signal is abnormal, the central door locking function will not work normally.

PROBABLE CAUSES



  • Malfunction of the central door lock switch (built into the power window main switch)
  • Malfunction of the door lock key cylinder
  • Malfunction of the ETACS-ECU
  • Damaged harness wires and connectors

DIAGNOSIS PROCEDURE


STEP 1. Connector check: F-02 power window main switch connector
