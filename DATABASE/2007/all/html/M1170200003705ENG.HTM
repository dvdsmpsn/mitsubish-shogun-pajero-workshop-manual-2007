Code No.21 Cancel Latch Signal System <6G7>


M11702000037ENG0000010001


DIAGNOSIS CODE SET CONDITIONS


The engine-A/T-ECU communicates cancellation retention information between the two microprocessors. This diagnosis code is set when cancellation retention information contains inconsistency.

PROBABLE CAUSE


Malfunction of the engine-A/T-ECU.

DIAGNOSIS PROCEDURE


Replace the engine-A/T-ECU (Refer to GROUP 13Aa - Engine-A/T-ECU ). Then check that diagnosis code No. 21 is not set.
