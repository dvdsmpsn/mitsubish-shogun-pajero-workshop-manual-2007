Code No. P0154: Left Bank Oxygen Sensor (Front) Circuit No Activity Detected


M11310100355ENG0000010000


OPERATION



  • Refer to Code No. P0150: Left bank oxygen sensor (front) circuit .
  • Refer to Code No. P0202: No. 2 injector system .
  • Refer to Code No. P0204: No. 4 injector system .
  • Refer to Code No. P0206: No. 6 injector system .

FUNCTION



  • The engine-A/T-ECU effects air/fuel ratio feedback control in accordance with the signals from the left bank oxygen sensor (front).
  • If the left bank oxygen sensor (front) has deteriorated, corrections will be made by the left bank oxygen sensor (rear).
  • Code No. P0154 becomes stored in memory if a failure is detected in the above air/fuel ratio feedback control system.

TROUBLE JUDGMENT


Check Conditions
  • 20 seconds or more have passed since the engine starting sequence was completed.
  • Engine coolant temperature is 7°C or more.
  • Engine speed is higher than 1,200 r/min.
  • Volumetric efficiency is 30 % or more.
  • Throttle position sensor output voltage is 4 V or less.
  • Except while fuel is being shut off.
  • Monitoring time: 30 seconds.
Judgment Criterion
  • Left bank oxygen sensor (front) output voltage does not get across 0.5 V within about 30 seconds.

PROBABLE CAUSES



  • Failed left bank oxygen sensor (front)
  • Exhaust leak
  • Incorrect fuel pressure
  • Failed engine-A/T-ECU
