Inspection Procedure 4: Auto-cruise Control cannot be Set <6G7>.


M11702000042ENG0000010002


COMMENTS ON TROUBLE SYMPTOM


The fail-safe function is probably cancelling auto-cruise control. In this case, checking the auto-cruise control system, MPI system and A/T system diagnosis codes. The M.U.T.-III can also be used to check if the circuits of each input switch are normal or not by checking the input switch codes.

PROBABLE CAUSES



  • Malfunction of the MPI system.
  • Malfunction of the A/T system.
  • Malfunction of the auto-cruise control switch.
  • Malfunction of the stop lamp switch.
  • Malfunction of the inhibitor switch.
  • Malfunction of the engine-A/T-ECU.

DIAGNOSIS PROCEDURE


STEP 1. Check the MPI system diagnosis code.


Refer to GROUP 13Ab - Diagnosis Function .

Q. Is any diagnosis code set?

Repair the MPI system. (Refer to GROUP 13Ab - Inspection Chart for Diagnosis Code ). Then go to Step 5.
Go to Step 2.

STEP 2. Check the A/T system diagnosis code.


Refer to GROUP 23A - Troubleshooting <A/T>, Diagnosis Function .

Q. Is any diagnosis code set?

Repair the A/T system (Refer to GROUP 23A - Troubleshooting <A/T>, Diagnosis Code Chart ). Then go to Step 5.
Go to Step 3.

STEP 3. Read the diagnosis code.


Read the auto-cruise control system diagnosis code (Refer to ).

Q. Is any diagnosis code set?

Diagnose the auto-cruise control system [Refer to , Check Chart for Diagnosis Codes <6G7>]. Then go to Step 5.
Go to Step 4.

STEP 4. Check the symptoms.


Q. Can auto-cruise control be set?

It can be assumed that this malfunction is intermittent (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points, How to Cope with Intermittent Malfunctions ).
Replace the engine-A/T-ECU (Refer to GROUP 13Aa - Engine-A/T-ECU ). Then go to Step 5.

STEP 5. Check the symptoms.


Q. Can auto-cruise control be set?

The procedure is complete.
Return to Step 1.
