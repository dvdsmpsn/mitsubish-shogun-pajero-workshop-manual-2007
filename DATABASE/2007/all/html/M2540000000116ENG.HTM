COMMUNICATION




ETACS-ECU connects to SWS*1 communication and CAN*2 communication to transmit signals mutually.
note*1: For details about SWS, refer to GROUP 54C, Smart Wiring System (SWS) .
note*2: For details about CAN, refer to GROUP 54D, Controller Area Network (CAN) .

BODY ELECTRICAL EQUIPMENT CONTROL FUNCTION


This function controls the following electrical equipment.

  • Buzzer
  • Central door locking system
  • Keyless entry system
  • Power window
  • Sunroof
  • Wiper, washer
  • Electric retractable remote controlled mirror
  • Exterior lamps
  • Interior lamps
