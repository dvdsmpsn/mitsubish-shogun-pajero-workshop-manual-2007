Code No.P1788 (P1751) A/T control relay system


M12301100916ENG0000010000




OPERATION


If a fail-safe operation is activated, the A/T control relay shuts off the power supply to the solenoid valve in accordance with the signal from the engine-A/T-ECU.

DIAGNOSIS CODE SET CONDITION


Code No.P1788 (P1751) will be set if the A/T control voltage is less than 7 V after the ignition switch is turned on.
If code No.P1788 (P1751) is set, the transmission will be held in 3rd gear.

PROBABLE CAUSES



  • Malfunction of A/T control relay
  • Damaged harness wires and connectors
  • Malfunction of the engine-A/T-ECU
