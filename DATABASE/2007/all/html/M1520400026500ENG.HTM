Code No.B1547: Cut Off For Passenger’s Front Air Bag


M15204000265ENG0000010000


cautionIf diagnosis code B1547 is set in the SRS-ECU, always diagnose the CAN main bus line.

DIAGNOSIS CODE SET CONDITIONS


When judging a collision, this code is set while the passenger's air bag is OFF.

DIAGNOSIS PROCEDURE


STEP 1. M.U.T.-III CAN bus diagnostics


Use the M.U.T.-III to diagnose the CAN bus lines.

Q. Is the check result normal?

Go to Step 2.
Repair the CAN bus line (Refer to GROUP 54D, Diagnosis ).

STEP 2. M.U.T.-III diagnosis code


Check that the diagnosis code except B1547 is set.
(1)Turn the ignition switch to the "ON" position.
(2)Check if the diagnosis code is set.
(3)Turn the ignition switch to the "LOOK" (OFF) position.

Q. Is the diagnosis code set?

Carry out the troubleshooting according to a set diagnosis code. Refer to .
Go to Step 3 .

STEP 3. Check whether the diagnosis code is reset.


Check again if the diagnosis code is set.
(1)Erase the diagnosis code.
(2)Ignition: "LOCK" (OFF) position to "ON"
(3)On completion, check that the diagnosis code is not reset.

Q. Is diagnosis code B1547 set?

Replace the SRS-ECU (Refer to ).
The procedure is complete.
