UPPER ARM BALL JOINT STARTING TORQUE CHECK


M13302000902ENG0000010000




1. After shaking the ball joint stud several times, install the nut to the stud and use special tool preload socket (MB990326) to measure the starting torque of the ball joint.
noteRotation starting torque measurement should be carried out with parts temperature above 0°C.
Standard value: 0.4 - 5.6 N·m
2. If the measured value exceeds the standard value, replace the upper arm ball joint assembly.
3. If the measured value is lower than the standard value, check that the upper arm ball joint turns smoothly without excessive play. If so, it is possible to use that ball joint.
