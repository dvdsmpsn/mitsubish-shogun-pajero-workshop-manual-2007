Code No.B1761: Chassies No. not recorded


M15410700012ENG0000010000


cautionWhenever the ECU is replaced, ensure that the input and output signal circuits are normal.

TROUBLE JUDGMENT


If the chassies No. data is not written when the immobilizer-ECU is replaced, the immobilizer-ECU sets diagnosis code No.B1761.

PROBABLE CAUSES



  • Chassies No. not written
  • Damaged harness wires and connectors
  • Malfunction of immobilizer-ECU

DIAGNOSIS PROCEDURE


Write the chassies No.

Write the chassies No. data to the immobilizer-ECU [Refer to GROUP 00 - How to perform vehicle identification number (chassies No. ) writing ].

Q. Is diagnosis code No.B1761 set?

Replace the immobilizer-ECU and register the ID codes (Refer to ).
The procedure is complete.
