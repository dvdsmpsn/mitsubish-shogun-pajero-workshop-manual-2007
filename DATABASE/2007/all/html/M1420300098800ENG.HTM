INSPECTION PROCEDURE 3: The Timer Lock Function does not Work After The Doors have been Unlocked by The Keyless Entry System.


M14203000988ENG0000010000


cautionWhenever the ECU is replaced, ensure that the input and output signal circuits are normal.

COMMENT ON TROUBLE SYMPTOM


If the keyless entry timer lock does not work normally, the input signal circuit(s) to the keyless entry transmitter or the ETACS-ECU may be defective.

POSSIBLE CAUSES



  • Malfunction of the keyless entry transmitter
  • Malfunction of the ETACS-ECU
  • Damaged harness wires and connectors

DIAGNOSIS PROCEDURE


STEP 1. Check the operation of the keyless entry system.


Check that the keyless entry system works normally.

Q. Is the check result normal?

Go to Step 2.
Refer to .

STEP 2. Customise function by using the SWS monitor.


"Timer relock after unlocking" is set to one of the followings by using SWS monitor customisation function. Check that the timer lock is activated according to the setting.

  • 30sec.
  • 60sec.
  • 120sec.
  • 180sec.

Q. Is the check result normal?

The trouble can be an intermittent malfunction (Refer to GROUP 00, How to Use Troubleshooting/Inspection Service Points - How to Cope with Intermittent Malfunction ).
Replace the ETACS-ECU.
