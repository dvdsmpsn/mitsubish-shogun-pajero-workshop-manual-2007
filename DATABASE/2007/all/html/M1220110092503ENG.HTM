Code No.32, 33: Transfer detection switch system


M12201100925ENG0000010000




OPERATION



  • System voltage is applied to the each detection switch (terminal No.1) from the transfer-ECU (terminal No.43, No.44, No.45, No.46 or No.47).
  • The each detection switch is grounded through the transfer case to the vehicle body.

DIAGNOSIS CODE SET CONDITIONS


In the case that the transfer shifting has not been completed during driving, the code No.32 is set as the open circuit or the short circuit of each transfer detection switch system, malfunction of the shift actuator or the malfunction of the transfer shift mechanism. In the case that the input signal from each transfer detection switch is unstable, the code No.33 is set as the open circuit or the short circuit of each transfer detection switch system.

PROBABLE CAUSES



  • Malfunction of the each transfer detection switch
  • Damaged harness wires and connectors
  • Malfunction of the transfer-ECU
  • Malfunction of the shift actuator
  • Malfunction of the transfer shift mechanism
