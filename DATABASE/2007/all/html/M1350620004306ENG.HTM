Code No.35: Engine System Fault
Code No.36: Engine-ECU Communication Error
Code No.37: A/T System Fault
Code No.38: A/T-ECU Communication Error
Code No.78: Engine ECU inappropriately installed


M13506200043ENG0000010000


cautionIf there is any problem in the CAN bus lines, an incorrect diagnosis code may be set. Diagnose the CAN bus lines before the diagnosis codes (Refer to GROUP 54D, CAN bus-line diagnostic flow ).

OPERATION


ASTC-ECU sends or receives data from engine-ECU <M/T>, engine-A/T-ECU <A/T>, combination meter, G and yaw rate sensor, and steering wheel sensor via the CAN bus line.

DIAGNOSIS CODE SET CONDITIONS


These diagnosis codes are set in the following cases:
Code No.35
  • An abnormality is found with the engine data received by ASTC-ECU. (In this case, although data is sent from the engine and A/T-ECU, an abnormality is present with the data content.)
Code No.36
  • All engine data from the engine and A/T-ECU are not sent to ASTC-ECU.
Code No.37
  • An abnormality is found with the A/T data received by ASTC-ECU. (In this case, although data is sent from the engine and A/T-ECU, an abnormality is present with the data content.)
Code No.38
  • All A/T data from the engine and A/T-ECU are not sent to ASTC-ECU.
Code No.78
  • The engine ECU is not installed properly.

PROBABLE CAUSES


The most likely causes for these diagnosis codes to set are:

  • Damaged wiring harness or connector
  • Malfunction of the Engine-ECU <M/T>
  • Malfunction of the Engine-A/T-ECU <A/T>
  • Malfunction of the ASTC-ECU

DIAGNOSIS


STEP 1. M.U.T.-III CAN bus diagnostics


cautionBefore connecting or disconnecting the M.U.T.-III, turn the ignition switch to the "LOCK" (OFF) position.
(1)Connect M.U.T.-III to the 16-pin diagnosis connector.
(2)Turn the ignition switch to the "ON" position.
(3)Diagnose the CAN bus line.
(4)Turn the ignition switch to the "LOCK" (OFF) position.

Q. Is the check result normal?

Go to Step 3.
Repair the CAN bus line (Refer to GROUP 54D, CAN bus line Diagnostic flow ). Then go to Step 2.

STEP 2. Check whether the diagnosis code is reset.


cautionBefore connecting or disconnecting the M.U.T.-III, turn the ignition switch to the "LOCK" (OFF) position.
(1)Turn the ignition switch to the "ON" position.
(2)Erase the diagnosis code.
(3)Turn the ignition switch to the "LOCK" (OFF) position.
(4)Turn the ignition switch to the "ON" position.
(5)Disconnect the M.U.T.-III.
(6)Drive the vehicle more than 10 seconds at the speed exceeds 30 km/h to judge the signals to the ASTC-ECU.
(7)Check if the diagnosis code is set.
(8)Turn the ignition switch to the "LOCK" (OFF) position.

Q. Is code No.35, 36, 37, 38 and 78 set?

Go to Step 3.
The procedure is complete.

STEP 3. Check for diagnosis codes of other systems.


Check if a diagnosis code is set from the engine-A/T-ECU <A/T> or the engine-ECU <M/T>.

Q. Is the check result normal?

Replace the ASTC-ECU. Then go to Step 4.
Diagnose the engine-ECU <M/T> and engine-A/T-ECU <A/T>.

STEP 4. Check whether the diagnosis code is reset.


cautionBefore connecting or disconnecting the M.U.T.-III, turn the ignition switch to the "LOCK" (OFF) position.
(1)Turn the ignition switch to the "ON" position.
(2)Erase the diagnosis code.
(3)Turn the ignition switch to the "LOCK" (OFF) position.
(4)Turn the ignition switch to the "ON" position.
(5)Disconnect the M.U.T.-III.
(6)Drive the vehicle more than 10 seconds at the speed exceeds 30 km/h to judge the signals to the ASTC-ECU.
(7)Check if the diagnosis code is set.
(8)Turn the ignition switch to the "LOCK" (OFF) position.

Q. Is code No.35, 36, 37, 38 and 78 set?

Repeat the troubleshooting from Step 1.
The procedure is complete.
