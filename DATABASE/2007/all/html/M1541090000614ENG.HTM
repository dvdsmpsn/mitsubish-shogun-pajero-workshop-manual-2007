HOW TO READ DIAGNOSIS CODE <HEADLAMP AUTOMATIC LEVELLING SYSTEM>



1. Using the special tool diagnosis code check harness (MB991529), earth diagnosis connector terminal No. 1 (diagnosis control terminal).
2. Turn the ignition switch to the ON position.
3. After elapsing 10 seconds or more, the headlamp automatic levelling warning lamp blinks. Then, read the diagnosis code.
noteDiagnosis code is stored until the ignition switch is turned to the "OFF" position.

Example of warning lamp blink when diagnosis code No. 24 is set
The warning lamp blink under normal conditions




HOW TO ERASE DIAGNOSIS CODE <HEADLAMP AUTOMATIC LEVELLING SYSTEM>


The diagnosis code can be erased by turning the ignition switch to the "OFF" position.
