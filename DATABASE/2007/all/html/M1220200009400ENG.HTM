DISASSEMBLY SERVICE POINT


<<A>> BALL BEARING REMOVAL



Use special tool Bearing remover (MD998801) to remove the ball bearing.

REASSEMBLY SERVICE POINTS


>>A<< BALL BEARING INSTALLATION



Use special tools to install the ball bearing.

  • Installer cap (MD998812)
  • Installer adapter (54) (MD998826)

>>B<< SNAP RING INSTALLATION



1. Install the thickest snap ring that can be fitted in the snap ring groove of the transfer input gear.
2. Make sure that the transfer input gear bearing end play meets the standard value.
Standard value: 0 - 0.06 mm
