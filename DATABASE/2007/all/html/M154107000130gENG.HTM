Code No.B1766: Transponder registered for other vehicle


M15410700013ENG0000010000


cautionWhenever the ECU is replaced, ensure that the input and output signal circuits are normal.

TROUBLE JUDGMENT


If the key registered for another vehicle is used to start the engine, the immobilizer-ECU sets diagnosis code No.B1766.

PROBABLE CAUSES



  • Ignition key (key ID) is another vehicle's key
  • Malfunction of immobilizer-ECU

DIAGNOSIS PROCEDURE


STEP 1. Check with another key that has already been registered.

Turn the ignition switch to the ON position using another key that has already been registered.

Q. Is the diagnosis code set?

Replace the all ignition key, and register the ID codes (Refer to ). After registering the ID codes, go to Step 2.
Replace the ignition key that the diagnosis code is set, and register the ID codes again (Refer to ). After registering the ID codes, go to Step 2.
STEP 2. Check whether the diagnosis code is reset.

Check again if the diagnosis code is set.
(1)Erase the diagnosis code.
(2)Turn the ignition switch to the "LOCK" (OFF) position once, and then turn it to the "ON" position again.
(3)Check that diagnosis code No.B1766 is not set.

Q. Is diagnosis code No.B1766 set?

Replace the immobilizer-ECU and register the ID codes (Refer to ).
The procedure is complete.
