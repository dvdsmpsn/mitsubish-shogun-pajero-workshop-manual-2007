Code No.13: Main relay system (Inside of ECU)


M12201100919ENG0000010000


OPERATION


The power supplied from the ignition switch is distributed to each part of ECU through the main relay located inside of ECU.

DIAGNOSIS CODE SET CONDITIONS


When the voltage at main relay OFF is 6 V and over, or the voltage at main relay ON is 6 V and less, the code No.13 is set as the main relay malfunction.

PROBABLE CAUSES



  • Malfunction of the transfer-ECU

DIAGNOSIS PROCEDURE


Replace the transfer-ECU.


(1)Replace the transfer-ECU.
(2)Test drive the vehicle.
(3)Check if the diagnosis code is set.

Q. Is diagnosis code No.13 set?

Replace the transfer-ECU.
The procedure is complete.
