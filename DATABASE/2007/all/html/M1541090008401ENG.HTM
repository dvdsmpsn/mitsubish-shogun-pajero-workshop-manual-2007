Code No. 24 Pitch Angle Abnormality


M15410900084ENG0000010000


cautionWhenever the ECU is replaced, ensure that the communication circuit is normal.

TROUBLE JUDGMENT


The headlamp automatic levelling-ECU receives a vehicle height signal from the front and rear height sensors and calculates the headlamp levelling pitch angle. If an abnormal pitch angle is calculated, diagnosis code No.24 is set.

COMMENTS ON TROUBLE SYMPTOM


The front height sensor, the rear height sensor or the headlamp automatic levelling-ECU may be defective.

PROBABLE CAUSES



  • Malfunction of the front height sensor
  • Malfunction of the rear height sensor
  • Malfunction of the headlamp automatic levelling-ECU
  • Damaged wiring harness and connectors
